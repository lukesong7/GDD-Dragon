## 2. 공성전 규칙

### 2.1 진행 흐름

```
[신청 기간]
  공격 혈맹: 공성 신청 (최대 5혈맹)
  수비 혈맹: 현재 성주 혈맹 + 동맹 혈맹
      ↓
[준비 기간 - 토요일 19:30~20:00]
  수비: 방어 시설 배치 (바리케이드, 포탑, 함정)
  공격: 성 외곽 집결
      ↓
[공성 Phase 1 - 외성 (30분)]
  공격측: 외성 문 파괴 또는 성벽 사다리 설치
  수비측: 외성 방어, 포탑 운용
      ↓
[공성 Phase 2 - 내성 (30분)]
  공격측: 내성 진입, 왕좌의 방 돌입
  수비측: 내성 사수, 최종 방어선
      ↓
[점령 판정]
  왕좌의 깃발을 3분간 점유한 혈맹이 새 성주
  시간 종료 시 수비 혈맹 유지
```

### 2.2 공성 병기

| 병기 | 사용측 | 효과 | 조건 |
|------|--------|------|------|
| 공성 망치 | 공격 | 성문 타격 (대폭 데미지) | 3인 운반 |
| 공성 사다리 | 공격 | 성벽 등반 가능 | 설치 시 20초 소요 |
| 공성 투석기 | 공격 | 원거리 범위 공격 | 조작 1인 + 장전 2인 |
| 화살 포탑 | 수비 | 자동 원거리 공격 | 미리 배치, 파괴 가능 |
| 끓는 기름 | 수비 | 성벽 아래 범위 데미지 | 성벽 위에서 사용 |
| 바리케이드 | 수비 | 통로 차단, 내구도 보유 | 미리 배치 |

> **[프로그래머 의견]** 공성 병기는 다수 플레이어가 동시에 인터랙션하는 객체다. 병기 조작 권한의 선점/해제 로직, 병기 위의 플레이어 동기화(사다리 등반 등)는 네트워크 코드에서 가장 까다로운 부분이 된다. 상태 동기화를 병기 전용 프로토콜로 분리하는 것을 권장.
>
> **[디자이너 의견]** 공성 병기의 스케일감이 중요하다. 투석기의 돌이 날아가며 성벽에 충돌하는 연출, 끓는 기름이 쏟아지는 이펙트 등이 공성전의 스펙터클을 만든다. 카메라 쉐이크와 함께.
>
> **[기획자 결정]** 모두 수용. 병기 시스템은 프로토타입 단계에서 우선 검증해야 할 핵심 피처. 연출은 '과하지 않되 확실한' 수준을 목표로 한다.

## 3. 성주 보상 및 운영

### 3.1 성주 혈맹 보상

| 보상 | 내용 |
|------|------|
| **세금 징수** | 성 등급에 따라 해당 지역 거래세 2~5% 자동 징수 |
| **전용 NPC 상점** | 성 내부 특별 NPC (희귀 소모품, 강화석 판매) |
| **사냥터 지배권** | 관할 사냥터 드롭률 +10% (혈맹원) |
| **성주 칭호** | 성주 전용 칭호 + 외형 이펙트 |
| **성 내부 시설** | 혈맹 전용 제작대, 창고, 부활 지점 |

### 3.2 세금 관리

- 세금은 자동 징수되어 **성 금고**에 축적
- 성 금고에서 인출 가능: 혈맹주, 부맹주
- 용도: 방어 시설 수리/강화, 혈맹 운영비, 자유 분배
- **세율 조정**: 성주가 0~5% 범위 내에서 조정 가능

> **[기획자 의견]** 세금 시스템은 성을 보유할 '이유'를 만드는 핵심이다. 세금이 너무 높으면 일반 유저 반발, 너무 낮으면 공성 동기 부족. 최대 5% 상한은 경제 파괴를 방지하면서 의미 있는 수입을 보장한다.

## 4. 공성전 밸런스

### 4.1 공수 밸런스

- **수비 유리 원칙**: 성의 지리적 이점으로 수비측이 기본적으로 유리
- **다수 공격 보정**: 공격 혈맹이 다수일 때 공격측 간 경쟁 발생 → 자연적 밸런스
- **공격측 보정**: 공격측 부활 지점을 성 근처에 배치하여 이동 시간 단축

### 4.2 혼전 방지

- 공격 혈맹이 3개 이상이면 공격측끼리도 PvP 가능
- 왕좌 점유는 한 혈맹만 가능 → 최종 국면에서 공격 혈맹 간 경쟁

> **[기획자 의견]** 공격 혈맹 간 혼전이 공성전의 백미다. 수비를 뚫은 뒤 왕좌를 차지하기 위한 '최후의 난전'이 리니지 공성전의 전설적인 순간을 만든다. 이 혼돈을 의도적으로 설계한다.

## 5. 영토전 (소규모)

공성전과 별도로, 사냥터 단위의 소규모 점령전.

### 5.1 영토전 규칙

| 항목 | 내용 |
|------|------|
| 개최 | 매일 21:00~22:00 |
| 대상 | 각 사냥터의 영토 깃발 |
| 참가 | 혈맹 Lv.4 이상 |
| 점령 | 깃발을 2분간 점유 |
| 유지 | 다음 영토전까지 유지 |

### 5.2 영토 보유 효과

- 해당 사냥터 드롭률 +10% (혈맹원)
- 비혈맹원 사냥터 이용 시 드롭 골드의 5%를 세금으로 자동 징수
- 영토 수비 NPC 5기 배치 가능

> **[프로그래머 의견]** 영토전은 매일 발생하므로 시스템 자동화가 중요. 영토전 시작/종료, 깃발 상태 관리, 보유 효과 적용/해제가 모두 자동으로 처리되어야 한다. 크론 스케줄러 기반으로 구현.
>
> **[기획자 결정]** 수용. 영토전은 코드 레벨에서 완전 자동화되며, 운영 개입 없이 매일 실행된다.
