# 몬스터 & 사냥터 시스템 (Monster & Hunting Ground)

- **프로젝트명**: Dragon
- **버전**: 1.0
- **최종 수정일**: 2026-03-23

---

## 1. 몬스터 분류

### 1.1 등급별 분류

| 등급 | 표시 | 체력 배율 | 출현 방식 | 전투 인원 |
|------|------|-----------|-----------|-----------|
| **일반 (Normal)** | 이름만 표시 | ×1 | 상시 리젠 | 솔로 |
| **강화 (Enhanced)** | 이름 + ★ | ×3 | 상시 리젠 (소수) | 솔로~2인 |
| **정예 (Elite)** | 금색 이름 | ×10 | 일정 주기 리젠 | 파티 (4~8인) |
| **필드 보스 (Field Boss)** | 붉은 이름 + 아이콘 | ×50 | 1~4시간 리젠 | 다수 파티 |
| **레이드 보스 (Raid Boss)** | 특수 UI | ×200 | 정해진 조건 | 레이드 (16~32인) |
| **드래곤 비스트** | 용 아이콘 | ×30~100 | 이벤트/조건 | 파티~레이드 |

### 1.2 속성별 분류

모든 몬스터는 5대 속성 중 하나를 보유:

| 속성 | 주요 서식지 | 특수 공격 패턴 |
|------|------------|---------------|
| 화 | 이그니스 화산지대, 용의 등뼈 산맥 | 화상 DoT, 범위 폭발 |
| 빙 | 글래시아 동토, 은빛 숲(일부) | 빙결, 둔화 |
| 지 | 철의 계곡, 아덴 평원 | 높은 방어력, 석화 |
| 풍 | 아스트라 고원, 흑풍 협곡 | 높은 회피, 넉백 |
| 암 | 엄브라 암흑지, 그림자 늪 | 흡혈, 저주, 시야 감소 |

## 2. 몬스터 AI

### 2.1 AI 등급

| AI 등급 | 적용 대상 | 행동 패턴 |
|---------|-----------|-----------|
| **단순** | 일반 몬스터 | 감지 → 접근 → 공격 반복 |
| **기본** | 강화 몬스터 | 단순 + HP 낮으면 도주/방어 자세 |
| **전술** | 정예 몬스터 | 어그로 관리, 스킬 패턴 사용, 취약 대상 우선 공격 |
| **고급** | 필드/레이드 보스 | 페이즈 전환, 다수 스킬 패턴, 기믹 보유 |
| **특수** | 드래곤 비스트 | 고급 + 환경 파괴, 속성 변환 |

### 2.2 보스 패턴 설계 원칙

- **페이즈 시스템**: HP 구간별 행동 패턴 전환 (100~70%, 70~30%, 30~0%)
- **텔레그래프**: 강력한 공격 전 반드시 시각적 예고 (1~2초 선딜)
- **기믹**: 단순 딜만으로 클리어 불가한 요소 (예: 특정 위치 이동, 오브젝트 파괴)
- **소환**: 보스가 주기적으로 졸개 소환, 파티 역할 분담 유도

> **[프로그래머 의견]** 보스 AI를 상태 머신(State Machine)이 아닌 행동 트리(Behavior Tree)로 구현하면 복잡한 패턴 조합이 훨씬 유연하다. 기획자가 비주얼 에디터에서 직접 보스 패턴을 조합할 수 있도록 툴을 만들면 개발 속도가 크게 향상된다.
>
> **[기획자 결정]** 수용. 보스 AI 비주얼 에디터는 개발 파이프라인 초기에 구축한다. 이 툴이 있으면 패턴 밸런싱과 신규 보스 추가가 빨라진다.

## 3. 사냥터 시스템

### 3.1 사냥터 설계 원칙

| 원칙 | 설명 |
|------|------|
| **레벨 구간별 배치** | 각 레벨 구간에 2~3개 사냥터 선택지 |
| **효율 차등** | 높은 효율 = 높은 위험 (PK 위험, 강한 몬스터) |
| **병목 설계** | 의도적으로 최적 사냥터를 제한 → 경쟁 유발 |
| **경로 다양성** | 같은 레벨이라도 안전/위험 루트 제공 |

### 3.2 사냥터 목록 (아르카디아 대륙)

| 사냥터 | 레벨 | 밀집도 | PK 위험도 | 핵심 드롭 |
|--------|------|--------|-----------|-----------|
| 루미나 초원 | 1~10 | 낮음 | 안전지대 | 기본 장비, 소비 |
| 고블린 야영지 | 8~15 | 중간 | 낮음 | 고급 무기 재료 |
| 은빛 숲 외곽 | 12~20 | 중간 | 낮음 | 엘프 제작 재료 |
| 은빛 숲 심층 | 18~25 | 높음 | 중간 | 희귀 약초, 엘프 장비 |
| 폐광산 | 22~32 | 높음 | 중간 | 광석, 드워프 장비 도면 |
| 아덴 평원 북부 | 30~40 | 중간 | 높음 | 영웅 장비 재료 |
| 아덴 평원 남부 | 35~45 | 높음 | 높음 | 영웅 장비 드롭 |
| 흑풍 협곡 입구 | 40~50 | 높음 | 매우 높음 | 고급 강화석 |
| 흑풍 협곡 심부 | 48~58 | 매우 높음 | 매우 높음 | 전설 재료 |
| 용의 등뼈 초입 | 50~60 | 중간 | 높음 | 드래곤 에센스(소) |
| 용의 등뼈 정상 | 58~68 | 높음 | 매우 높음 | 드래곤 에센스(중) |
| 그림자 늪 | 55~65 | 중간 | 높음 | 암속성 재료, 암살자 장비 |
| 폐허의 전장 | 60~72 | 매우 높음 | 최고 | 전설 장비, PvP 보상 |
| 용의 심장 외곽 | 68~80 | 높음 | 최고 | 드래곤 에센스(대) |
| 용의 심장 핵심 | 78~90 | 매우 높음 | 최고 | 신화 재료, 전설 스킬북 |

### 3.3 사냥터 경쟁 메카닉

> **[기획자 의견]** 사냥터 경쟁은 리니지의 핵심이다. 최적 사냥터를 '부족하게' 설계하여 혈맹 간 갈등을 자연스럽게 유발하는 것이 핵심이다.

- **채널 시스템**: 사냥터당 3~5개 채널, 채널 이동 쿨다운 10분
- **점유 보너스**: 같은 사냥터에서 30분 이상 연속 사냥 시 드롭률 +5%
- **경쟁 알림**: 다른 혈맹원이 같은 사냥터 진입 시 알림
- **영토 사냥터**: 혈맹이 점령한 사냥터는 점령 혈맹에 드롭률 +10%, 비점령 혈맹 -5%

## 4. 드롭 테이블

### 4.1 드롭 구조

```
몬스터 처치
├── 확정 드롭: 골드 (100%)
├── 일반 드롭 테이블: 소비/재료 (등급별 확률)
├── 장비 드롭 테이블: 무기/방어구 (등급별 확률)
└── 특수 드롭 테이블: 스킬북/에센스/이벤트 아이템 (극저확률)
```

### 4.2 드롭 확률 기준 (일반 몬스터)

| 드롭 종류 | 확률 |
|-----------|------|
| 골드 | 100% (몬스터 레벨에 비례) |
| 일반 소비/재료 | 30~50% |
| 고급 재료 | 5~10% |
| 일반 장비 | 3~5% |
| 고급 장비 | 0.5~1% |
| 희귀 장비 | 0.05~0.1% |
| 영웅 장비 | 0.001% |

- 정예 몬스터: 일반 대비 드롭 확률 ×5, 영웅 장비 드롭 가능
- 필드 보스: 영웅 장비 확정 1개 + 전설 재료 확률 드롭
- 레이드 보스: 전설 장비 확정 1~2개 + 신화 재료 확률 드롭

### 4.3 드롭 권한

| 방식 | 적용 상황 |
|------|-----------|
| **최다 피해** | 필드 보스 — 가장 많은 데미지를 입힌 파티 |
| **파티 분배** | 파티 사냥 — 설정에 따라 |
| **레이드 분배** | 레이드 보스 — 레이드장 분배 또는 주사위 |
| **개인 드롭** | 일반 몬스터 — 마지막 타격자 |

> **[프로그래머 의견]** 필드 보스에서 '최다 피해' 방식은 힐러가 불리하다. 데미지 기여뿐 아니라 힐/버프 기여도를 포함한 '기여도 점수' 시스템으로 확장해야 힐러가 파티에서 환영받는다.
>
> **[기획자 결정]** 수용. 기여도 = 딜링(60%) + 힐링(25%) + 탱킹(15%). 기여도가 가장 높은 파티에 드롭 권한 부여.

## 5. 리젠 시스템

| 몬스터 등급 | 리젠 시간 | 특이사항 |
|------------|----------|----------|
| 일반 | 30~60초 | 채널별 독립 |
| 강화 | 1~3분 | |
| 정예 | 10~30분 | 출현 시 지역 공지 |
| 필드 보스 | 1~4시간 (±30분 랜덤) | 서버 전체 공지 |
| 레이드 보스 | 고정 스케줄 또는 조건 트리거 | 서버 전체 공지 + 사전 연출 |
| 드래곤 비스트 | 동적 이벤트 트리거 | 서버 전체 경보 |

> **[프로그래머 의견]** 필드 보스 리젠 시간에 랜덤 편차를 주는 것은 '보스 캠핑'을 완전히 방지하지는 못하지만, 캠핑의 비용(시간)을 높여 건전한 경쟁을 유도한다. 리젠 타이머는 서버 내부에서만 관리하고 클라이언트에 절대 노출하지 않아야 한다.
>
> **[기획자 결정]** 수용. 추가로 필드 보스가 일정 시간 내 처치되지 않으면 자동 소멸(리젠 타이머 리셋)하여 한 세력이 보스를 '잠금(locking)'하는 행위를 방지한다.
