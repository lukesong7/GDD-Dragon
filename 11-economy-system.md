# 경제 시스템 (Economy System)

- **프로젝트명**: Dragon
- **버전**: 1.0
- **최종 수정일**: 2026-03-23

---

## 1. 재화 구조

### 1.1 재화 종류

| 재화 | 획득 방법 | 용도 | 거래 가능 |
|------|-----------|------|-----------|
| **골드 (Gold)** | 몬스터 드롭, 퀘스트, 거래 | 모든 인게임 거래, NPC 상점, 강화 | 가능 |
| **다이아 (Diamond)** | 캐시 구매, 일부 이벤트 | 캐시샵 전용 | 불가 |
| **명예 포인트** | PvP, 공성전 | PvP 장비, PvP 소모품 | 불가 |
| **혈맹 자금** | 혈맹 활동, 영토 세금 | 혈맹 시설, 혈맹 스킬 | 불가 |
| **드래곤 에센스** | 보스, 레이드, 공성전 | 각성, 전설/신화 제작 | 가능 |

### 1.2 골드 유통 구조

```
[유입]                              [소멸]
몬스터 드롭 ──────┐          ┌────── 장비 강화 실패
퀘스트 보상 ──────┤          ├────── 수리비
NPC 아이템 판매 ──┤  [순환]  ├────── NPC 상점 구매
                  ├──유저──→├────── 거래소 수수료 (3%)
                  │  경제  │├────── 텔레포트 비용
                  │        ├────── 영토/공성 세금
                  │        └────── 제작 비용
```

> **[프로그래머 의견]** 골드 유입/소멸 비율은 서버 운영의 핵심 지표다. 유입 > 소멸이면 인플레이션, 소멸 > 유입이면 디플레이션이 발생한다. 실시간 모니터링 대시보드에서 일일 유입량, 소멸량, 총 골드량, 거래량을 추적해야 한다.
>
> **[기획자 결정]** 수용. 목표 비율은 유입:소멸 = 1:0.85~0.95. 인플레이션 발생 시 운영 파라미터(수리비, 강화 비용)를 동적 조정하는 '경제 밸런서' 시스템을 설계한다.

## 2. 거래소 시스템

### 2.1 거래소 기본

| 항목 | 내용 |
|------|------|
| 이용 NPC | 왕도 + 각 주요 마을 |
| 수수료 | 판매 금액의 3% |
| 등록 제한 | 동시 10개 아이템 |
| 등록 기간 | 최대 48시간 |
| 최저/최고가 | 시세 기반 ±30% 범위 제한 |
| 검색 | 아이템명, 등급, 카테고리, 옵션 필터 |

### 2.2 시세 시스템

- 최근 7일 거래 내역 기반 평균 시세 표시
- 시세 그래프 제공 (가격 추이)
- 비정상 거래 감지: 시세 대비 300% 이상/30% 이하 거래 자동 차단

> **[프로그래머 의견]** 시세 시스템은 RMT(현금거래) 방지의 핵심이다. 비정상 거래 패턴(반복적 고가 구매, 특정 계정 간 일방적 거래)을 AI 기반으로 탐지하는 시스템을 운영 툴에 포함해야 한다.
>
> **[기획자 결정]** 수용. 비정상 거래 탐지 시스템은 런칭 필수 기능으로, 개발 Phase 3에 포함한다.

## 3. 개인 상점

### 3.1 개인 상점 기본

| 항목 | 내용 |
|------|------|
| 개설 조건 | Lv.15 이상 |
| 상점 형태 | 판매 상점 / 구매 상점 |
| 최대 등록 | 8개 아이템 |
| 상점 메시지 | 30자 커스텀 간판 |
| 오프라인 상점 | 프리미엄 패스 유저만 가능 |

### 3.2 개인 상점 특수 규칙

- 안전지대(마을)에서만 개설 가능
- 상점 개설 중 이동/전투 불가
- 오프라인 상점: 로그아웃 후에도 상점 유지 (최대 24시간)

> **[디자이너 의견]** 마을에 개인 상점이 밀집하면 시각적 혼란이 발생한다. 상점 캐릭터를 약간 반투명 처리하거나, 상점 밀집 지역에서 LOD를 적극 적용해야 한다.
>
> **[기획자 결정]** 수용. 추가로 마을 내 '상점 거리' 지정 구역을 만들어 상점 개설을 유도하고, 구역 외 개설 시 상점 노출 우선순위를 낮춘다.

## 4. NPC 상점

### 4.1 기본 NPC 상점

| 상점 | 판매 물품 |
|------|-----------|
| 잡화상 | 물약, 귀환석, 텔레포트 스크롤, 화살 |
| 무기상 | 일반/고급 등급 무기 |
| 방어구상 | 일반/고급 등급 방어구 |
| 마법상 | 일반 스킬북, 버프 주문서 |

### 4.2 특수 NPC 상점

| 상점 | 위치 | 재화 | 판매 물품 |
|------|------|------|-----------|
| PvP 상인 | 전장 NPC | 명예 포인트 | PvP 장비, PvP 소모품 |
| 용사냥꾼 상점 | 용의 등뼈 산맥 | 드래곤 에센스 | 각성 재료, 드래곤 스킬북 |
| 성주 상점 | 성 내부 | 골드 | 희귀 소모품 (성주 혈맹 전용) |
| 암시장 | 엄브라 암흑지 | 골드 | 성향치 회복 물약, 금지 아이템 |

## 5. 경제 밸런스 설계

### 5.1 골드 가치 기준

| 기준 항목 | 수치 | 비고 |
|-----------|------|------|
| Lv.50 캐릭터 시간당 평균 수입 | 10,000골드 | 기준 사냥터 |
| HP 물약(대) 가격 | 500골드 | 시간당 20개 소비 기준 |
| 영웅 장비 거래소 시세 (목표) | 500,000~2,000,000골드 | 50~200시간 수입 |
| 전설 장비 거래소 시세 (목표) | 5,000,000~20,000,000골드 | 500~2000시간 수입 |

### 5.2 인플레이션 방지 장치

| 장치 | 설명 |
|------|------|
| 강화 비용 | 단계별 골드 소모 (주요 소멸 장치) |
| 거래소 수수료 | 3% 고정 (대량 거래 시 큰 소멸) |
| 수리비 | 사망/내구도 소모 시 골드 소멸 |
| 세금 | 영토/성주 세금 (일부 성 금고에 축적 후 재분배) |
| 텔레포트 비용 | 거리에 비례한 골드 소모 |
| 제작 비용 | NPC 제작대 사용료 |
| 스탯/스킬 리셋 | 리셋 시 골드 소모 |

> **[기획자 의견]** 경제 시스템의 핵심은 '골드가 항상 부족하다'는 느낌을 유지하는 것이다. 골드가 넘치면 성취감이 사라지고, 부족하면 더 사냥하고, 더 거래하고, 더 경쟁한다. 인플레이션은 게임 경제를 파괴하는 1순위 위협이다.
