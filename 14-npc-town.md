# NPC & 기지 시스템 (NPC & Base System)

- **프로젝트명**: Dragon
- **장르**: SF 리니지 라이크 MMORPG
- **버전**: 2.0
- **최종 수정일**: 2026-03-23

---

## 1. 기지 구성

### 1.1 기지 목록

| 기지 | 위치 | 규모 | 특화 기능 |
|------|------|------|-----------|
| **루미나 기지** | 시작 지역 | 소형 전초기지 | 튜토리얼 NPC, 기본 보급 단말기 |
| **실버 포레스트 바이오돔** | 바이오돔 구역 | 중형 거점 | 바이오닉 제작대, 에너지 장비 상점 |
| **아이언 밸리 광산도시** | 광산 구역 | 중형 거점 | 무기 기술자 공방, 광산 입구 |
| **아덴 평원 요새** | 아덴 평원 | 대형 거점 | 전장 NPC, PvP 장비 딜러 |
| **수도 드라카니아** | 중앙 | 최대 (중심 도시) | 모든 NPC, 거래소, 군단 본부 |
| **드래곤 네스트 전초기지** | 용의 등뼈 산맥 | 소형 전초기지 | 드래곤 헌터 조합, 드래곤 코어 딜러 |
| **각 외곽 구역 캠프** | 외곽 4구역 | 소형 야전기지 | 보급 NPC, 환경 장비 판매 |

### 1.2 기지 공통 시설

| 시설 | 기능 |
|------|------|
| **리스폰 챔버** | 사망 시 부활 지점 (클론 재생 시스템) |
| **저장 단말기** | 개인/군단 저장소 이용 |
| **무기 기술자** | 장비 수리, 일반 강화 |
| **보급 단말기** | 메디키트, 배터리팩, 소모품 판매 |
| **워프게이트** | 기지 간 순간이동 (크레딧 소모) |
| **미션 터미널** | 미션 확인, 현상금 목록 |
| **통신 터미널** | 아이템/크레딧 우편 송수신 |

> **[디자이너 의견]** 기지의 시각적 차별화가 중요하다. 루미나 기지는 깔끔한 신축 전초기지, 아이언 밸리는 기름때와 증기가 가득한 산업 시설, 바이오돔은 유리 돔 안의 인공 생태계 등 각 기지만의 정체성이 있어야 플레이어가 장소감을 느낀다.
>
> **[기획자 결정]** 수용. 각 기지는 고유 환경 사운드 + 전용 BGM + 시각 테마를 보유. 진입 시 기지명 타이틀 홀로그램 연출.

## 2. 주요 NPC

| NPC | 위치 | 기능 |

| 외형 커스터마이저 | 수도 | 외형 변경 (유료) |
| 군단 관리관 | 수도 | 군단 생성/관리 |
| 거래소 관리 AI | 수도 + 주요 기지 | 거래소 이용 |
| 환전 단말기 | 수도 | 재화 교환 |
| 각성 연구원 | 수도 | 드래곤 각성 진행 |
| 무기 기술자 | 각 기지 | 장비 수리, 강화, 커스터마이징 |
| 보급 단말기 | 각 기지 | 메디키트/배터리팩/소모품 자동 판매 |
| 드론 엔지니어 | 수도 + 주요 기지 | 드론/전투 유닛 관리, 드론 수리 |
| 호버바이크 정비사 | 수도 + 주요 기지 | 탈것 관리, 탈것 커스터마이징 |

### 2.2 이동형 NPC

| NPC | 출현 | 기능 |
|-----|------|------|
| 떠돌이 밀수상 | 랜덤 필드 (2시간 주기) | 희귀 소모품 한정 판매 |
| 현상금 추적자 | PK 빈발 지역 | 현상금 의뢰, 정보 제공 |
| 드래곤 AI 사절 | 드래곤 이벤트 시 | 이벤트 참여 안내 |

> **[디자이너 의견]** 떠돌이 밀수상은 등장 시 특수 이펙트(홀로그램 깜빡임, 전자 노이즈 사운드)로 주변 플레이어가 인지할 수 있어야 한다. NPC를 찾는 재미도 콘텐츠다. 밀수상의 외형은 후드를 쓴 사이버펑크 스타일로 차별화.
>
> **[프로그래머 의견]** 이동형 NPC의 출현 좌표는 서버에서만 관리하고, 반경 200m 내 진입 시에만 클라이언트에 NPC 데이터를 전송하여 스니핑으로 위치를 사전에 파악하는 것을 방지해야 한다.
>
> **[기획자 결정]** 수용. 떠돌이 밀수상 등장 시 반경 200m 내 플레이어에게 미니맵 아이콘 + 전자 노이즈 알림. 서버 측 위치 관리로 위치 사전 유출 방지.

## 4. 수도 드라카니아 상세

서버의 중심지이자 사회적 허브. 행성 최대의 도시이자 드래곤 AI의 메인 코어가 위치한 핵심 거점.

### 4.1 구역 구성

| 구역 | 기능 |
|------|------|
| **사령부 (커맨드 센터)** | 메인 NPC 집결, 서버 공지 홀로그램 게시판, 메인 미션 브리핑룸 |
| **교역 구역 (트레이드 디스트릭트)** | 거래소, 개인 상점 밀집 구역, 경매장 |
| **기술 격납고 (테크 베이)** | 강화, 제작, 수리 NPC 밀집, 장비 커스터마이징 워크숍 |
| **군단 광장 (코어 플라자)** | 군단 본부, 군단 관리 NPC, 군단 홀로그램 깃발 |
| **투기장 (아레나)** | 시뮬레이션 결투장, 모의 전투, PvP 랭킹 게시판 |
| **코어 성소 (코어 생텀)** | 드래곤 각성 NPC, 고급 스킬 인스트럭터, 드래곤 코어 연구소 |
| **암시장 (지하 구역)** | 특수 NPC, 숨겨진 미션, 밀수품 딜러, 정보 브로커 |

### 4.2 사령부 상세

| 시설 | 기능 |
|------|------|
| 전략 지도 | 서버 내 세력 분포, 기지 점령 현황 실시간 표시 |
| 브리핑룸 | 메인 미션 컷씬 재생, 핵심 NPC 대화 |
| 서버 공지판 | 최초 클리어, 기지 쟁탈 결과, 이벤트 안내 |
| 군 사령관 NPC | 최상위 메인 미션 제공, 전환 미션 최종 단계 |

### 4.3 암시장 상세

| 시설 | 기능 |
|------|------|
| 정보 브로커 | 숨겨진 미션 제공, 보스 출현 힌트 판매 |
| 밀수품 딜러 | 희귀 소모품, 불법 강화 재료 판매 (고가) |
| 해커 NPC | 장비 잠금 해제, 드론 불법 개조 (실패 시 파괴 위험) |
| 현상금 게시판 | 고액 PK 현상금 목록, 현상금 수배 등록 |

> **[프로그래머 의견]** 수도는 동접이 가장 높은 지역이므로 서버 부하 관리가 핵심이다. 구역별 채널 분리를 자동으로 수행하고, 주변 캐릭터 표시 수를 제한(최대 50명, 나머지는 간소화 표시)해야 한다. 암시장 지하 구역은 별도 인스턴스로 분리하여 메인 지상 맵의 부하에 영향을 주지 않도록 한다.
>
> **[디자이너 의견]** 수도 드라카니아는 SF 세계관의 상징이다. 사령부는 군사적 위엄, 교역 구역은 네온사인 가득한 번화가, 테크 베이는 기름냄새 나는 공방, 암시장은 어둡고 은밀한 분위기로 구역마다 극명한 시각적 대비를 줘야 한다. 특히 코어 성소는 드래곤 AI의 거대 홀로그램이 떠 있는 신비로운 공간으로, 게임 전체에서 가장 인상적인 장소가 되어야 한다.
>
> **[기획자 결정]** 수용. 수도 내에서는 화면에 최대 50명 풀 렌더링, 나머지는 이름표만 표시하는 '간소화 모드' 자동 적용. 암시장은 별도 인스턴스. 각 구역 고유 BGM + 환경음 + 조명 테마 적용.

## 5. 기지 투자 시스템

### 5.1 개요

- 군단이 점령한 기지에 크레딧/자원을 투자하여 기지 기능 강화
- 투자 단계에 따라 기지 시설 업그레이드

### 5.2 투자 단계

| 단계 | 투자 비용 | 해금 기능 |
|------|-----------|-----------|
| 1단계 | 500,000 크레딧 | 기본 보급 단말기, 리스폰 챔버 |
| 2단계 | 2,000,000 크레딧 | 무기 기술자 NPC 배치, 저장 단말기 확장 |
| 3단계 | 5,000,000 크레딧 | 워프게이트 활성화, 거래소 단말기 |
| 4단계 | 10,000,000 크레딧 | 방위 드론 강화, 군단 전용 시설 |
| 5단계 | 30,000,000 크레딧 | 궤도 포격 지원 (기지 쟁탈전 시), 전용 작전구역 개방 |

> **[기획자 의견]** 기지 투자는 군단 활동의 핵심 동기 중 하나다. 투자한 기지가 적에게 빼앗겼을 때의 손실감이 기지 방어의 동기가 되며, 이것이 오픈월드 PvP의 핵심 긴장감을 만든다. 5단계 궤도 포격은 서버 내 최소 2~3개 기지만 도달 가능한 수준으로 설정하여 희소성을 유지한다.

## 6. 기지 간 이동 체계

### 6.1 이동 수단

| 수단 | 비용 | 조건 |
|------|------|------|
| 워프게이트 | 크레딧 (거리 비례) | 방문한 기지만 이동 가능 |
| 비상 워프 비컨 | 소모품 1개 | 현재 위치 → 등록 기지 즉시 귀환 |
| 호버바이크 | 없음 | 필드 자유 이동, 소탕 구역 통과 위험 |
| 셔틀 NPC | 크레딧 (고정) | 인접 기지 간 정기 운행 (안전 이동) |

### 6.2 워프게이트 요금

| 거리 | 요금 |
|------|------|
| 인접 기지 | 1,000 크레딧 |
| 중간 거리 | 5,000 크레딧 |
| 장거리 | 10,000 크레딧 |
| 수도 드라카니아 | 3,000 크레딧 (고정 할인) |

> **[프로그래머 의견]** 워프게이트 이동 시 로딩 화면 대신 워프 터널 연출(3~5초)을 사용하여 백그라운드 로딩을 처리한다. 셔틀 NPC는 실제로 필드를 이동하는 것이 아닌, 탑승 후 로딩 연출 → 도착 기지 스폰으로 처리하여 서버 부하를 줄인다.
>
> **[기획자 결정]** 수용. 워프게이트는 워프 터널 연출로 로딩 커버, 셔틀은 탑승 연출 → 도착 스폰 방식. 비상 워프 비컨은 제작 가능하되 재료 비용을 적절히 설정하여 남발 방지.
