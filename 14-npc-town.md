## 2. 주요 NPC

### 2.1 기능형 NPC

| NPC | 위치 | 기능 |
|-----|------|------|
| 전직 관리자 | 왕도 + 각 주요 마을 | 전직 퀘스트 제공 |
| 스킬 마스터 | 각 마을 | 스킬 습득/리셋 |
| 미용사 | 왕도 | 외형 변경 (유료) |
| 혈맹 관리자 | 왕도 | 혈맹 생성/관리 |
| 거래소 관리자 | 왕도 + 주요 마을 | 거래소 이용 |
| 환전상 | 왕도 | 재화 교환 |
| 각성 사제 | 왕도 | 드래곤 각성 진행 |

### 2.2 이동형 NPC

| NPC | 출현 | 기능 |
|-----|------|------|
| 떠돌이 상인 | 랜덤 필드 (2시간 주기) | 희귀 소모품 한정 판매 |
| 현상금 사냥꾼 | PK 빈발 지역 | 현상금 의뢰, 정보 제공 |
| 용의 사절 | 드래곤 이벤트 시 | 이벤트 참여 안내 |

> **[디자이너 의견]** 떠돌이 상인은 등장 시 특수 이펙트(작은 불꽃, 종소리)로 주변 플레이어가 인지할 수 있어야 한다. NPC를 찾는 재미도 콘텐츠다.
>
> **[기획자 결정]** 수용. 떠돌이 상인 등장 시 반경 200m 내 플레이어에게 미니맵 아이콘 + 효과음 알림.

## 4. 왕도 드라카니아 상세

서버의 중심지이자 사회적 허브.

### 4.1 구역 구성

| 구역 | 기능 |
|------|------|
| **왕궁 광장** | 메인 NPC 집결, 서버 공지 게시판 |
| **상인의 거리** | 거래소, 개인 상점 밀집 구역 |
| **대장간 골목** | 강화, 제작, 수리 NPC 밀집 |
| **혈맹 광장** | 혈맹 홀, 혈맹 관리 NPC |
| **투기장** | 결투장, 모의 전투 |
| **각성의 성소** | 드래곤 각성 NPC, 고급 스킬 마스터 |
| **암시장 (지하)** | 특수 NPC, 숨겨진 퀘스트 |

> **[프로그래머 의견]** 왕도는 동접이 가장 높은 지역이므로 서버 부하 관리가 핵심이다. 구역별 채널 분리를 자동으로 수행하고, 주변 캐릭터 표시 수를 제한(최대 50명, 나머지는 간소화 표시)해야 한다.
>
> **[기획자 결정]** 수용. 왕도 내에서는 화면에 최대 50명 풀 렌더링, 나머지는 이름표만 표시하는 '간소화 모드' 자동 적용.
