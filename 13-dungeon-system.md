# 던전 시스템 (Dungeon System)

- **프로젝트명**: Dragon
- **버전**: 1.0
- **최종 수정일**: 2026-03-23

## 1. 던전 분류

### 1.1 던전 유형

| 유형 | 인원 | 입장 제한 | 특징 |
|------|------|-----------|------|
| **솔로 던전** | 1인 | 일 3회 | 레벨업 보조, 재료 수급 |
| **파티 던전** | 4~8인 | 일 2회 | 역할 분담, 영웅 장비 드롭 |
| **레이드** | 16~32인 | 주 1회 | 보스 기믹, 전설 장비 드롭 |
| **혈맹 던전** | 혈맹원 20~40인 | 주 1회 | 혈맹 보상, 혈맹 경험치 |
| **무한 던전 (천공의 탑)** | 1~4인 | 무제한 | 층 클리어 랭킹, 점진적 난이도 |

### 1.2 던전 목록

| 던전 | 유형 | 레벨 | 위치 | 핵심 보상 |
|------|------|------|------|-----------|
| 고블린 동굴 | 솔로/파티 | 15~25 | 고블린 야영지 | 고급 장비, 제작 재료 |
| 폐광산 심층부 | 파티 | 25~35 | 철의 계곡 | 광석, 드워프 장비 도면 |
| 은빛 숲 성역 | 파티 | 30~40 | 은빛 숲 | 엘프 장비, 마법 재료 |

| 용의 둥지 | 파티 | 50~60 | 용의 등뼈 산맥 | 드래곤 에센스, 영웅 장비 |
| 얼어붙은 왕좌 | 레이드 | 65~75 | 글래시아 동토 | 전설 장비, 빙속성 각성 재료 |

| 암흑의 왕좌 | 레이드 | 80~90 | 엄브라 암흑지 | 전설/신화 재료, 최종 보스 |
| 천공의 탑 | 무한 | 80+ | 왕도 드라카니아 | 층별 보상, 랭킹 |

## 2. 던전 메카닉

### 2.1 인스턴스 구조

- 각 파티/레이드에 독립 인스턴스 생성
- 던전 내부 PvP 불가 (PK 방지)
- 제한 시간: 파티 던전 30~60분, 레이드 90~120분
- 전멸 시: 입구 부활 (시간은 계속 진행)

### 2.2 보스 기믹 단계

| 난이도 | 적용 던전 | 기믹 예시 |
|--------|-----------|-----------|
| **기본** | 솔로, 저레벨 파티 | 텔레그래프 회피, 기본 패턴 반복 |
| **중급** | 중레벨 파티 | 페이즈 전환, 졸개 소환, 장판 회피 |
| **상급** | 고레벨 파티 | 역할 분담 기믹 (탱커 스위칭, 힐러 체크) |
| **최상** | 레이드 | 다중 기믹 동시 처리, DPS 체크, 즉사기 |

### 2.3 레이드 보스 예시: 얼어붙은 왕좌 — 빙룡 글래시어

```
Phase 1 (100~70% HP): 기본 패턴
- 아이스 브레스: 전방 60도 원뿔 범위, 빙결 3초
- 아이스 스파이크: 랜덤 3지점에 빙기둥, 3초 후 폭발
- 프로스트 오라: 주기적 전체 둔화

Phase 2 (70~30% HP): 얼음 분열
- 보스가 3개 분신으로 분열 (본체 찾아서 공격)
- 가짜 분신 공격 시 반사 데미지
- 바닥이 점차 빙판으로 변함 (이동 시 미끄러짐)

Phase 3 (30~0% HP): 빙하 폭풍
- 전체 데미지 + 빙결 (힐러 체크)
- 안전 지대가 주기적으로 변경 (이동 강제)
- 광폭화: 10분 경과 시 공격력 300% 증가 (DPS 체크)
```

> **[프로그래머 의견]** 레이드 보스의 페이즈 전환은 서버에서만 판정하고 클라이언트에 상태를 동기화해야 한다. 특히 분신 기믹에서 '본체 판별'이 클라이언트 데이터로 유출되면 치팅이 가능하므로, 분신의 HP/반응을 동일하게 보여주되 서버에서만 본체 위치를 관리해야 한다.
>
> **[디자이너 의견]** 빙판 미끄러짐 메카닉은 컨트롤 재미와 짜증의 경계가 얇다. 미끄러짐을 약간의 관성으로 표현하되, 조작 불능까지는 가지 않도록 해야 한다.
>
> **[기획자 결정]** 수용. 빙판은 이동 시 관성 +30% 수준, 조작은 가능하되 정밀 포지셔닝이 어려운 수준. 방빙 장비/버프로 대응 가능하게 설계하여 준비의 가치를 부여.

## 3. 천공의 탑 (무한 던전)

### 3.1 구조

- 1층부터 무한히 진행
- 5층마다 보스 층
- 10층마다 체크포인트 (다음 도전 시 해당 층부터 시작)
- 난이도는 층마다 점진적으로 상승

### 3.2 랭킹

| 랭킹 | 기준 | 보상 |
|------|------|------|
| 솔로 랭킹 | 최고 도달 층 + 클리어 시간 | 주간 보상 (직업별) |
| 파티 랭킹 | 최고 도달 층 + 클리어 시간 | 주간 보상 (파티) |
| 시즌 랭킹 | 시즌 내 최고 기록 | 시즌 한정 보상 |

### 3.3 층별 보상

| 층 | 보상 |
|----|------|
| 매 층 | 골드 + 경험치 (층 비례) |
| 10층마다 | 장비 상자 (등급 점진 상승) |
| 50층 | 영웅 장비 확정 |
| 100층 | 전설 장비 선택 상자 |
| 100층+ | 랭킹 보상 재료, 칭호 |

## 4. 던전 난이도 모드

파티/레이드 던전은 난이도 선택 가능.

| 난이도 | 몬스터 강화 | 보상 배율 | 추가 기믹 |
|--------|------------|-----------|-----------|
| 일반 | ×1 | ×1 | 기본 |
| 영웅 | ×2 | ×2.5 | 기믹 추가 |
| 전설 | ×4 | ×5 | 기믹 추가 + 제한시간 단축 |

> **[기획자 의견]** 난이도 선택제는 같은 던전을 다른 경험으로 만들어 콘텐츠 수명을 늘린다. 전설 난이도 클리어는 서버 공지로 띄워 도전 의욕을 자극한다.
