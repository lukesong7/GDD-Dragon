# 인스턴스 작전 시스템 (Instance Operation System)

- **프로젝트명**: Dragon
- **장르**: SF 리니지 라이크 MMORPG
- **버전**: 2.0
- **최종 수정일**: 2026-03-23

## 1. 작전구역 분류

### 1.1 작전구역 유형

| 유형 | 인원 | 입장 제한 | 특징 |
|------|------|-----------|------|
| **솔로 작전** | 1인 | 일 3회 | 레벨업 보조, 재료 수급 |
| **파티 작전** | 4~8인 | 일 2회 | 역할 분담, 영웅 장비 드롭 |
| **레이드 작전** | 16~32인 | 주 1회 | 보스 기믹, 전설 장비 드롭 |
| **군단 작전** | 군단원 20~40인 | 주 1회 | 군단 보상, 군단 경험치 |
| **무한 작전 (궤도 타워)** | 1~4인 | 무제한 | 층 클리어 랭킹, 점진적 난이도 |

## 2. 작전 메카닉

### 2.1 인스턴스 구조

- 각 파티/레이드에 독립 인스턴스 생성
- 작전구역 내부 PvP 불가 (PK 방지)
- 제한 시간: 파티 작전 30~60분, 레이드 작전 90~120분
- 전멸 시: 입구 리스폰 (시간은 계속 진행)

### 2.2 보스 기믹 단계

| 난이도 | 적용 작전구역 | 기믹 예시 |
|--------|---------------|-----------|
| **기본** | 솔로, 저레벨 파티 | 텔레그래프 회피, 기본 패턴 반복 |
| **중급** | 중레벨 파티 | 페이즈 전환, 증원 유닛 소환, 장판 회피 |
| **상급** | 고레벨 파티 | 역할 분담 기믹 (탱커 스위칭, 힐러 체크) |
| **최상** | 레이드 | 다중 기믹 동시 처리, DPS 체크, 즉사기 |

### 2.3 레이드 보스 예시: 빙결 커맨드 센터 — GLACIA 전투 AI

```
Phase 1 (100~70% HP): 기본 프로토콜
- 크라이오 캐논: 전방 60도 원뿔 범위, 빙결 3초
- 아이스 드론 사출: 랜덤 3지점에 냉각 드론 투하, 3초 후 폭발
- 프로스트 필드: 주기적 전체 둔화 (냉각 가스 살포)

Phase 2 (70~30% HP): 분열 프로토콜
- GLACIA가 3개 홀로그램 분신으로 분열 (본체 판별 후 집중 공격)
- 가짜 분신 공격 시 전자기 반사 데미지
- 바닥에 냉각제가 점차 확산 (이동 시 미끄러짐)

Phase 3 (30~0% HP): 빙하 프로토콜 최종 단계
- 냉동 폭풍: 전체 데미지 + 빙결 (힐러 체크)
- 안전 구역이 주기적으로 변경 (열원 장치 주변만 안전, 이동 강제)
- 오버클럭: 10분 경과 시 공격력 300% 증가 (DPS 체크)
```

> **[프로그래머 의견]** 레이드 보스의 페이즈 전환은 서버에서만 판정하고 클라이언트에 상태를 동기화해야 한다. 특히 분신 기믹에서 '본체 판별'이 클라이언트 데이터로 유출되면 치팅이 가능하므로, 분신의 HP/반응을 동일하게 보여주되 서버에서만 본체 위치를 관리해야 한다. GLACIA AI의 홀로그램 이펙트는 GPU 파티클 시스템으로 처리하여 다수 플레이어 환경에서도 프레임 저하를 최소화한다.
>
> **[디자이너 의견]** 냉각제 미끄러짐 메카닉은 컨트롤 재미와 짜증의 경계가 얇다. 미끄러짐을 약간의 관성으로 표현하되, 조작 불능까지는 가지 않도록 해야 한다. SF 세계관답게 열원 장치를 활성화하여 안전 구역을 만드는 협동 기믹은 전략적 깊이를 더한다.
>
> **[기획자 결정]** 수용. 냉각제 구간은 이동 시 관성 +30% 수준, 조작은 가능하되 정밀 포지셔닝이 어려운 수준. 방빙 장갑/버프로 대응 가능하게 설계하여 준비의 가치를 부여. 열원 장치 활성화는 엔지니어 계열 클래스에 유리한 보조 기믹으로 배치.

## 3. 궤도 타워 (무한 작전)

### 3.1 구조

- 궤도 엘리베이터 형태의 수직 구조물, 1층부터 무한히 상승
- 5층마다 보스 층 (각 층의 보스는 과거 전투 AI 프로토타입)
- 10층마다 체크포인트 (다음 도전 시 해당 층부터 시작)
- 난이도는 층마다 점진적으로 상승
- 상위 층으로 갈수록 우주 환경 효과 추가 (무중력 구간, 방사능 구역 등)

### 3.2 랭킹

| 랭킹 | 기준 | 보상 |
|------|------|------|
| 솔로 랭킹 | 최고 도달 층 + 클리어 시간 | 주간 보상 (직업별) |
| 파티 랭킹 | 최고 도달 층 + 클리어 시간 | 주간 보상 (파티) |
| 시즌 랭킹 | 시즌 내 최고 기록 | 시즌 한정 보상 |

### 3.3 층별 보상

| 층 | 보상 |
|----|------|
| 매 층 | 크레딧 + 경험치 (층 비례) |
| 10층마다 | 장비 컨테이너 (등급 점진 상승) |
| 50층 | 영웅 장비 확정 |
| 100층 | 전설 장비 선택 컨테이너 |
| 100층+ | 랭킹 보상 재료, 칭호 |

> **[프로그래머 의견]** 궤도 타워의 무한 확장 구조는 절차적 생성(Procedural Generation)으로 구현해야 서버 메모리를 절약할 수 있다. 10층 단위로 시드를 고정하여 동일 시즌 내 같은 층은 동일한 레이아웃을 보장하되, 시즌마다 시드를 교체하여 신선함을 유지한다.
>
> **[기획자 결정]** 수용. 궤도 타워는 시즌제로 운영하며, 시즌마다 층 레이아웃 갱신 + 특수 변이 층(50층마다 보스 러시, 100층마다 환경 재해 층) 추가.

## 4. 작전구역 난이도 모드

파티/레이드 작전은 난이도 선택 가능.

| 난이도 | 적 강화 | 보상 배율 | 추가 기믹 |
|--------|---------|-----------|-----------|
| 일반 | ×1 | ×1 | 기본 |
| 영웅 | ×2 | ×2.5 | 기믹 추가 |
| 전설 | ×4 | ×5 | 기믹 추가 + 제한시간 단축 |

### 4.1 전설 난이도 특수 규칙

- 메디키트/배터리팩 사용 횟수 제한 (파티당 총 30회)
- 리스폰 횟수 제한 (파티당 총 3회)
- 추가 환경 위험 요소 (방사능 누출, 중력 이상, 전자기 간섭)
- 첫 클리어 시 서버 전체 공지 + 한정 칭호 부여

> **[디자이너 의견]** 전설 난이도의 환경 위험 요소는 SF 세계관의 특색을 살리는 좋은 기회다. 중력 이상 구간에서는 점프 궤적이 달라지고, 전자기 간섭 구간에서는 HUD가 노이즈 처리되어 미니맵이 비활성화되는 등 시각/체감적으로 차별화된 경험을 제공할 수 있다.
>
> **[기획자 결정]** 수용. 난이도 선택제는 같은 작전구역을 다른 경험으로 만들어 콘텐츠 수명을 늘린다. 전설 난이도 클리어는 서버 공지로 띄워 도전 의욕을 자극한다. 환경 위험 요소는 전설 난이도 전용으로 배치.

## 5. 작전구역 탐색 및 진입

### 5.1 진입 방식

| 방식 | 설명 |
|------|------|
| 현장 진입 | 작전구역 입구 NPC에게 직접 진입 |
| 워프게이트 진입 | 수도 드라카니아의 중앙 워프게이트에서 진입 (크레딧 소모) |
| 자동 매칭 | 파티 작전 자동 매칭 시스템 (역할별 매칭) |
| 군단 소집 | 군단장이 군단 작전 소집 시 일괄 진입 |

### 5.2 귀환 규칙

- 작전 완료 시: 비상 워프 비컨으로 기지 귀환
- 작전 포기 시: 10초 캐스팅 후 입구로 강제 퇴장
- 연결 끊김 시: 3분 내 재접속하면 현재 위치 유지, 초과 시 입구로 이동

> **[프로그래머 의견]** 자동 매칭 시스템은 역할(탱커/딜러/힐러) 비율을 강제하되, 대기 시간이 3분을 초과하면 비율 제한을 완화하여 매칭 속도를 우선시해야 한다. 서버 간 교차 매칭도 지원하여 과소 서버의 대기 시간 문제를 해결한다.
>
> **[기획자 결정]** 수용. 자동 매칭: 기본 역할 비율 강제(3분) → 비율 완화(5분) → 교차 서버 매칭(7분). 매칭 대기 중 다른 활동 가능.
