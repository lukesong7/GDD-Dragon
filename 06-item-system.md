# 아이템 시스템 (Item System)

- **프로젝트명**: Dragon
- **버전**: 2.0
- **장르**: SF 리니지 라이크 MMORPG
- **최종 수정일**: 2026-03-23

---

## 1. 아이템 분류

### 1.1 대분류

| 분류 | 설명 | 거래 가능 여부 |
|------|------|----------------|
| **장비** | 무기, 파워아머, 모듈, 액세서리 | 대부분 가능 (일부 귀속) |
| **소비** | 메디키트, 배터리팩, 부스터 칩 | 가능 |
| **재료** | 제작/강화 재료 | 가능 |
| **퀘스트** | 퀘스트 전용 아이템 | 불가 |
| **기타** | 워프게이트 코드, 비상 워프 비컨 등 | 가능 |

### 1.2 아이템 등급

| 등급 | 색상 | 드롭 확률 (일반 호스틸) | 특징 |
|------|------|----------------------|------|
| **일반 (Normal)** | 흰색 | 높음 | 기본 스탯만 존재 |
| **고급 (Uncommon)** | 녹색 | 중간 | 추가 옵션 1개 |
| **희귀 (Rare)** | 파란색 | 낮음 | 추가 옵션 2개 |
| **영웅 (Hero)** | 보라색 | 매우 낮음 | 추가 옵션 3개 + 세트 효과 |
| **전설 (Legend)** | 주황색 | 보스/레이드 전용 | 추가 옵션 4개 + 고유 효과 |
| **신화 (Mythic)** | 붉은색 | 제작 전용 | 추가 옵션 5개 + 고유 효과 + 외형 변환 |

> **[디자이너 의견]** 등급별 색상은 업계 표준을 따르되, 아이템 아이콘 테두리와 이름 색상을 동시에 적용해야 인벤토리에서 한눈에 구분된다. 전설 이상은 아이콘에 에너지 파동 애니메이션을 추가하여 수집 욕구를 자극해야 한다. SF 테마에 맞게 홀로그래픽 반짝임을 적용한다.
>
> **[기획자 결정]** 수용. 추가로 신화 등급은 획득 시 서버 전체 공지를 띄워 희소성을 부각한다.

## 2. 장비 시스템

### 2.1 장비 슬롯

| 슬롯 | 위치 | 비고 |
|------|------|------|
| 주무기 | 오른손 | 클래스별 무기 제한 |
| 보조 장비/에너지 실드 | 왼손 | 에너지 실드/보조무기/없음 |
| 헬멧 | 머리 | HUD 기능 연동 |
| 상체 아머 | 상체 | 파워아머 코어 부위 |
| 하체 아머 | 하체 | 파워아머 구동 부위 |
| 전투 글러브 | 손 | 사격/근접 정밀도 보정 |
| 전투 부츠 | 발 | 이동/부스터 관련 |
| 백팩 유닛 | 등 | 에너지 공급/추가 장갑 |
| 데이터 링크 | 목 | 통신/버프 관련 |
| 센서 유닛 | 귀 | 2개 슬롯, 탐지/회피 관련 |
| 전술 링 | 손가락 | 2개 슬롯, 나노테크 보정 |
| 유틸리티 벨트 | 허리 | 메디키트/배터리팩 자동 사용 슬롯 연동 |

### 2.2 사이버네틱 임플란트 슬롯 (추가)

| 슬롯 | 위치 | 비고 |
|------|------|------|
| 신경 임플란트 | 두뇌 | 시전 속도/반응 속도 보정 |
| 근섬유 강화 | 근육 | STR/물리 공격력 보정 |
| 안구 임플란트 | 눈 | 크리티컬/명중률 보정 |
| 나노 코어 | 심장 | HP/EP 최대치 보정 |

- 사이버네틱 임플란트는 영웅 등급 이상에서만 존재
- 장착 시 귀속, 해제 시 파괴
- 나노 합성으로만 제작 가능

### 2.3 장비 옵션

#### 기본 옵션 (등급 무관, 모든 장비 보유)
- 무기: 공격력, 공격 속도
- 파워아머: 장갑, HP 보너스
- 모듈/액세서리: 스탯 보너스

#### 추가 옵션 (랜덤 부여)

| 옵션 | 수치 범위 (영웅 등급 기준) |
|------|--------------------------|
| STR/DEX/CON/INT/WIS/LUK | +3~10 |
| 크리티컬 확률 | +1~5% |
| 크리티컬 데미지 | +5~20% |
| 공격 속도 | +3~10% |
| 이동 속도 | +3~8% |
| HP/EP 추가 | +100~500 |
| PvP 공격력/방어력 | +2~8% |
| 속성 공격/저항 (플라즈마/전기/냉각/중력) | +5~15 |
| 드롭률 증가 | +1~5% |
| 경험치 증가 | +1~5% |

> **[프로그래머 의견]** 추가 옵션이 완전 랜덤이면 '최적 옵션 조합'을 위한 무한 파밍이 발생한다. 옵션 재설정(리롤) 시스템을 제공하되, 재화 소모를 통해 경제 소멸 장치로 활용하면 좋다.
>
> **[기획자 결정]** 수용. '나노 재구성 키트' 아이템으로 추가 옵션 1개를 랜덤 재설정 가능. '정밀 나노 재구성 키트'로 원하는 옵션 카테고리를 지정하고 수치만 랜덤. 둘 다 제작/거래 가능.

### 2.4 세트 장비

영웅 등급 이상 장비는 세트를 구성할 수 있다.

| 세트 피스 수 | 보너스 예시 |
|-------------|-----------|
| 2개 | 스탯 +5 |
| 4개 | 고유 효과 (예: 피격 시 10% 확률 에너지 배리어 생성) |
| 6개 (풀세트) | 극강 효과 (예: 스킬 쿨다운 -15%) |

#### 세트 장비 예시

| 세트명 | 클래스 | 구성 | 풀세트 효과 |
|--------|--------|------|------------|
| 타이탄 프로토콜 | 가디언 | 파워아머 6부위 | 피격 시 20% 확률 에너지 배리어(HP 15% 흡수), 장갑 +20% |
| 팬텀 스트라이크 | 에이전트 | 파워아머 6부위 | 은폐 시 이동속도 감소 없음, 기습 데미지 +40% |
| 코어 오버로드 | 테크노서 | 파워아머 6부위 | 에너지/테크 공격 데미지 +25%, EP 소모 -15% |
| 발할라 메딕 | 메딕 | 파워아머 6부위 | 회복량 +30%, 전장 소생 쿨타임 -50% |

## 3. 강화 시스템

### 3.1 일반 강화 (+1 ~ +20)

| 강화 단계 | 성공 확률 | 실패 시 | 효과 |
|-----------|----------|---------|------|
| +1 ~ +3 | 100% | - | 기본 스탯 +3%/단계 |
| +4 ~ +6 | 80% | 강화 수치 유지 | 기본 스탯 +4%/단계 |
| +7 ~ +9 | 50% | 강화 -1 | 기본 스탯 +5%/단계 + 에너지 글로우 이펙트 |
| +10 ~ +12 | 30% | 강화 -2 | 기본 스탯 +6%/단계 + 에너지 오라 이펙트 |
| +13 ~ +15 | 15% | 강화 -3 또는 장비 파괴(10%) | 기본 스탯 +8%/단계 + 플라즈마 이펙트 |
| +16 ~ +20 | 5~10% | 강화 -3 또는 장비 파괴(30%) | 기본 스탯 +10%/단계 + 최상위 에너지 폭발 이펙트 |

### 3.2 강화 보호 아이템

| 아이템 | 효과 | 획득처 |
|--------|------|--------|
| 부스터 칩 (보호형) | 실패 시 강화 수치 유지 | 나노 합성, 프리미엄샵 |
| 안전 프로토콜 칩 | 실패 시 장비 파괴 방지 | 이벤트, 프리미엄샵 |
| 확률 증폭 칩 | 성공 확률 +5~10% | 나노 합성, 이벤트 |

> **[프로그래머 의견]** 강화 시스템은 서버에서만 확률 판정을 수행해야 한다. 클라이언트는 연출만 담당. 확률 판정 로그는 전수 기록하여 운영 툴에서 조회 가능하게 해야 CS 대응이 가능하다.
>
> **[디자이너 의견]** 강화 연출이 매우 중요하다. +10 이상 시도 시 카메라가 줌인되며 에너지가 장비에 집중되는 연출, 성공하면 에너지 폭발 + 전기 방전 연출, 실패하면 에너지 역류 + 장비 흔들림. 파괴 시에는 장비가 과부하로 분해되는 연출이 필요하다. SF 테마에 맞게 홀로그래픽 수치 카운팅 + 전자음으로 긴장감을 조성한다.
>
> **[기획자 결정]** 모두 수용. 강화 연출은 단계별로 긴장감이 점증하도록 설계하며, +15 이상 성공 시 서버 전체 공지를 띄운다.

### 3.3 속성 부여

- 무기에 4대 속성(플라즈마/전기/냉각/중력) 중 하나를 부여 가능
- **드래곤 코어** 소모
- 속성 상성에 따라 대상에 추가 데미지 또는 감소
- 속성 부여 시 무기 이펙트가 해당 속성 색상으로 변경

## 4. 제작 시스템

### 4.1 제작 카테고리

| 카테고리 | 제작 가능 아이템 | 핵심 재료 |
|----------|-----------------|-----------|
| **무기공방 (무기/파워아머)** | 영웅~신화 등급 장비 | 합금, 나노섬유, 드래곤 코어 |
| **나노 합성 (소비)** | 메디키트, 부스터 칩, 나노 재구성 키트 | 나노 물질, 에너지 결정, 호스틸 부산물 |
| **모듈 제작 (액세서리/임플란트)** | 전술 링, 데이터 링크, 사이버네틱 임플란트 | 마이크로 칩, 드래곤 코어, 희귀 광물 |

### 4.2 제작 성공률

| 등급 | 성공률 | 실패 시 |
|------|--------|---------|
| 영웅 | 60% | 재료 50% 반환 |
| 전설 | 30% | 재료 30% 반환 |
| 신화 | 10% | 재료 20% 반환 |

- 메카닉 종족 특성: 제작 성공률 +10%
- 제작 숙련도: 반복 제작으로 성공률 점진적 증가 (최대 +15%)

> **[기획자 의견]** 신화 등급 장비는 제작으로만 획득 가능하다. 이는 메카닉 종족과 제작 전문 유저에게 경제적 가치를 부여하고, 제작 콘텐츠 자체를 엔드게임 활동으로 만드는 장치다.
>
> **[프로그래머 의견]** 제작 시스템의 재료 확인 → 확률 판정 → 결과물 생성/재료 반환은 반드시 하나의 트랜잭션으로 처리해야 한다. 중간에 서버 크래시가 나면 재료만 사라지는 최악의 CS 케이스가 발생한다.
>
> **[기획자 결정]** 수용. 제작 전 과정은 서버 측 원자적 트랜잭션으로 처리하며, 실패 시 롤백 보장.

## 5. 인벤토리 및 창고

### 5.1 인벤토리

| 항목 | 수치 |
|------|------|
| 기본 슬롯 | 40칸 |
| 최대 확장 | 100칸 (프리미엄샵 확장 모듈) |
| 무게 시스템 | 있음 (STR에 따라 최대 적재량 증가) |
| 무게 초과 시 | 이동 속도 감소, 일정 이상 이동 불가 |

### 5.2 창고

| 항목 | 수치 |
|------|------|
| 개인 창고 | 100칸 (기지 NPC 이용) |
| 군단 창고 | 200칸 (권한별 접근) |
| 창고 확장 | 프리미엄샵 확장 모듈로 +50칸씩 |

> **[프로그래머 의견]** 무게 시스템은 유저가 직관적으로 이해하기 어려울 수 있다. 인벤토리 UI에 현재 적재량/최대 적재량 게이지를 상시 표시하고, 임계치(80%, 100%) 도달 시 경고를 띄워야 한다.
>
> **[디자이너 의견]** 적재량 게이지는 인벤토리 하단에 바(bar) 형태로 표시. 80% 이상이면 노란색 경고등, 100% 이상이면 빨간색 과적 경고로 변경. SF 테마에 맞게 디지털 게이지 + 경고 사운드를 적용한다.
>
> **[기획자 결정]** 수용.

## 6. 특수 아이템

### 6.1 드래곤 코어

| 등급 | 획득처 | 용도 |
|------|--------|------|
| 드래곤 코어(소) | 작전 구역 Lv.50+ 호스틸 | 속성 부여, 일반 강화 재료 |
| 드래곤 코어(중) | 필드 보스, 던전 | 고급 강화, 제작 재료 |
| 드래곤 코어(대) | 레이드 보스 | 신화 제작, 드래곤 스킬 습득 |
| 드래곤 코어(순수) | 특수 이벤트, 최상위 레이드 | 신화 장비 각성, 최종 강화 |

### 6.2 비상 워프 비컨

- 사용 시 등록한 기지/거점으로 즉시 이동
- 전투 중 사용 불가 (비전투 전환 후 3초 시전)
- 기본 지급 아이템, 소비 시 상점에서 재구매 가능
- 군단 기지 전용 비상 워프 비컨 별도 존재

### 6.3 워프게이트 코드

- 각 기지/거점 간 워프게이트 이동에 필요
- 한 번 방문한 거점은 자동 등록
- 워프게이트 사용 시 크레딧 소모
