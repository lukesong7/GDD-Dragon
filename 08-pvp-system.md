# PvP 시스템 (PvP System)

- **프로젝트명**: Dragon
- **장르**: SF 리니지 라이크 MMORPG
- **버전**: 2.0
- **최종 수정일**: 2026-03-23

---

## 1. PvP 기본 구조

### 1.1 PvP 유형

| 유형 | 설명 | 성향치 영향 | 패널티 |
|------|------|------------|--------|
| **PK (Player Kill)** | 비동의 공격. 필드에서 타 플레이어를 일방적으로 공격 | 감소 (악화) | 사망 시 장비 드롭, 경험치 손실 |
| **정당방위** | PK 공격에 대한 반격 | 없음 | 경미 |
| **결투** | 동의 하에 1:1 또는 팀 대전 | 없음 | 없음 |
| **군단전** | 군단 간 선전포고 후 전투 | 없음 | 없음 |
| **기지 쟁탈전** | 기지/스테이션 점령을 위한 대규모 전투 | 없음 | 없음 |
| **영토전** | 작전 구역 점령을 위한 소규모 전투 | 없음 | 없음 |
| **전장(배틀필드)** | 인스턴스 PvP | 없음 | 없음 |

> **[기획자 의견]** PK에만 패널티를 부여하고, 나머지 PvP는 페널티 프리로 만드는 것이 핵심이다. PK는 '가능하지만 대가가 따르는' 행위, 나머지 PvP는 '적극 권장하는' 행위로 구분한다.

## 2. 성향치 시스템

### 2.1 성향치 구조

```
성향치 범위: -1000 (극악) ~ 0 (중립) ~ +1000 (선)

[수호자]  [선량]  [중립]  [약탈자]  [악인]  [극악]
+1000   +500     0     -200    -500   -1000
```

### 2.2 성향치 변동

| 행동 | 성향치 변화 |
|------|-----------|
| PK (먼저 공격하여 살해) | -50 ~ -100 (대상 레벨 차에 따라) |
| 정당방위 킬 | +5 |
| 몬스터 소탕 | +1/100마리 |
| 퀘스트 완료 | +5~20 |
| PK 플레이어 처치 (성향치 음수인 자) | +20 |
| 악인 현상금 처치 | +50 |

### 2.3 성향치에 따른 효과

| 성향 구간 | 효과 |
|-----------|------|
| **수호자 (+800~+1000)** | 사망 시 패널티 -50%, NPC 할인 10%, 전용 칭호 |
| **선량 (+200~+799)** | 사망 시 패널티 -20% |
| **중립 (-199~+199)** | 기본 |
| **약탈자 (-200~-499)** | 거점 보안 드론에게 공격받음, NPC 이용 제한 |
| **악인 (-500~-799)** | 보안 드론 공격 강화, 사망 시 장비 드롭 확률 +10%, 현상금 대상 |
| **극악 (-800~-1000)** | 보안 드론 즉시 사살, 사망 시 장비 드롭 확률 +30%, 이름 붉은색 |

> **[디자이너 의견]** 성향치에 따라 캐릭터 이름 색상이 변해야 한다. 선 = 하늘색, 중립 = 흰색, 약탈자 = 주황색, 악인/극악 = 빨간색. 이름만 봐도 위험 인물을 식별할 수 있어야 한다.
>
> **[프로그래머 의견]** 성향치 음수 유저가 거점에 진입할 때 보안 드론의 어그로 범위와 공격력을 동적으로 조정해야 한다. 성향치가 낮을수록 보안 드론의 감지 범위 확대 + 데미지 증가.
>
> **[기획자 결정]** 모두 수용. 보안 드론은 성향치 -200 이하 감지 범위 30m, -500 이하 50m, -800 이하 거점 진입 즉시 감지.

## 3. PK 시스템 상세

### 3.1 PK 규칙

- **PK 가능 조건**: Lv.30 이상, 안전지대(거점, 튜토리얼 존) 밖
- **PK 보호**: Lv.30 미만 캐릭터는 PK 공격 불가
- **PK 선언**: 공격 시 2초간 '조준' 모션 (기습 방지, 반응 시간 보장)
- **PK 플래그**: PK 공격 후 5분간 보라색 이름 (정당방위 대상)
- **다수 PK**: 연속 PK 시 성향치 하락 가속 (2번째 PK부터 ×1.5배)

### 3.2 PK 피해자 보호

- PK 당한 플레이어: **경험치/장비 손실 없음**
- 부활 시 10초 무적 (도주 기회)
- PK 신고: GM에게 자동 로그 전송

> **[기획자 의견]** PK 피해자의 손실을 0으로 만들고, PK 가해자에게만 패널티를 집중하는 구조다. 이렇게 하면 PK가 '게임 파괴'가 아닌 '도전과 긴장감'으로 느껴진다. PK를 당해도 돌아오면 되고, PK를 하면 스스로 리스크를 감수해야 한다.

## 4. 결투 시스템

### 4.1 결투 규칙

| 항목 | 내용 |
|------|------|
| 신청 | 대상 우클릭 → 결투 신청 |
| 수락 | 10초 내 수락/거절 |
| 결투장 | 현재 위치에 반경 30m 에너지 배리어 결투장 생성 |
| 승리 조건 | HP 0 또는 결투장 이탈 |
| 패널티 | 없음 (HP 1로 생존) |
| 관전 | 주변 플레이어 관전 가능 |

### 4.2 팀 결투

- 2:2, 3:3, 5:5 팀 결투 지원
- 파티 리더끼리 신청
- 팀 전원 사망 시 패배

## 5. 전장(배틀필드)

### 5.1 전장 종류

| 전장 | 인원 | 규칙 | 보상 |
|------|------|------|------|
| **우주 투기장 (Space Arena)** | 1:1 | 개인 랭킹전, 3전 2선승 | PvP 포인트 + 랭킹 |
| **드래곤 코어 쟁탈전** | 15:15 | 중앙 드래곤 코어 확보, 일정 점수 달성 | PvP 장비 재료 |
| **전술 전장 (Tactical Ground)** | 30:30 | 3개 거점 점령, 20분 제한 | 대량 PvP 포인트 |

### 5.2 전장 보상

- **PvP 포인트**: 전장 참여/승리로 획득, PvP 전용 장비 구매
- **PvP 시즌**: 3개월 단위, 시즌 종료 시 랭킹별 보상
- **PvP 등급**: 브론즈 → 실버 → 골드 → 다이아몬드 → 마스터 → 그랜드마스터 → 드래곤

| 등급 | PvP 포인트 요구 | 보상 |
|------|----------------|------|
| 드래곤 (최상위) | 상위 0.1% | 시즌 한정 전설 무기 외형, 칭호, 에너지 윙 이펙트 |
| 그랜드마스터 | 상위 1% | 영웅 PvP 장비 세트, 칭호 |
| 마스터 | 상위 5% | 영웅 PvP 장비, 칭호 |

> **[프로그래머 의견]** 전장 매칭에서 PvP 등급 기반 매칭이 필수. 매칭 대기 시간이 길어지면 등급 범위를 점진적으로 확장하되, 2단계 이상 차이나는 매칭은 방지해야 한다.
>
> **[디자이너 의견]** 전장 입장/승리 시 전용 BGM이 중요하다. 입장 시 긴장감 있는 일렉트로닉 BGM, 승리 시 승리 팡파레, 패배 시 담담한 BGM으로 감정 곡선을 설계해야 한다.
>
> **[기획자 결정]** 모두 수용. 매칭은 ±1등급 범위에서 시작, 2분 경과 시 ±2등급으로 확장. BGM은 전장 유형별 별도 제작.

## 6. 현상금 시스템

- 성향치 -500 이하(악인)인 플레이어에게 자동 현상금 부여
- 현상금 = 악인 플레이어의 PK 횟수 × 100크레딧 (NPC 기금)
- 누구든 현상금 대상을 처치하면 현상금 수령 + 성향치 +50
- 현상금 목록은 현상금 터미널 NPC에서 조회 가능

> **[기획자 의견]** 현상금 시스템은 PK 생태계를 자정하는 장치다. 일방적 PK가 서버 분위기를 해치지 않도록, PK를 많이 하면 자연스럽게 '사냥감'이 되는 구조를 만든다.
