# 미션 시스템 (Mission System)

- **프로젝트명**: Dragon
- **장르**: SF 리니지 라이크 MMORPG
- **버전**: 2.0
- **최종 수정일**: 2026-03-23

---

## 1. 미션 분류

### 1.1 미션 유형

| 유형 | 설명 | 반복 | 보상 규모 |
|------|------|------|-----------|
| **메인 미션** | 세계관 스토리 진행, 핵심 시스템 해금 | 1회 | 대 |
| **전환 미션** | 1차/2차 전환 조건 | 1회 | 전환 해금 |
| **사이드 미션** | 지역/NPC 관련 보조 스토리 | 1회 | 중 |
| **일일 미션** | 매일 갱신, 기본 활동 보상 | 매일 (최대 5개) | 소 |
| **주간 미션** | 주간 갱신, 중간 난이도 | 매주 (최대 3개) | 중 |
| **군단 미션** | 군단 단위 수행 | 매주 | 군단 보상 |
| **업적 미션** | 장기 목표 (숨김 포함) | 1회 | 칭호/외형 |

### 1.2 메인 미션 챕터 구성

| 챕터 | 레벨 | 제목 | 핵심 내용 |
|------|------|------|-----------|
| 1장 | 1~10 | 기동의 시작 | 튜토리얼, 드래곤블러드 나노머신 발현 |
| 2장 | 10~20 | 바이오돔의 비밀 | 실버 포레스트 바이오돔 세력과 첫 동맹, 드래곤 코어 발견 |
| 3장 | 20~30 | 강철의 울림 | 아이언 밸리 광산도시와 기술 제휴, 광산 구역의 호스틸 위협 |
| 4장 | 30~45 | 황야의 전쟁 | 첫 PvP 도입, 세력 간 자원 쟁탈 |
| 5장 | 45~60 | 드래곤 AI의 그림자 | 드래곤 비스트(변이체) 대량 출현, 2차 전환 |
| 6장 | 60~75 | 세력의 충돌 | 군단 활동, 기지 쟁탈전 도입 |
| 7장 | 75~99 | 시조 AI의 유산 | 외곽 구역 탐사, 드래곤 코어 쟁탈 최종 레이드 |

> **[디자이너 의견]** 메인 미션의 컷씬은 핵심 장면에만 집중하되, 스킵 가능해야 한다. 특히 2회차 이상 플레이어(전생, 서브 캐릭터)를 위해 전체 스킵 옵션 필수. SF 세계관의 홀로그램 브리핑 연출은 시각적 임팩트를 높이되 길이는 30초 이내로 유지한다.
>
> **[프로그래머 의견]** 챕터 전환 시 대규모 맵 로딩이 발생할 수 있으므로, 컷씬 재생 중 백그라운드 스트리밍 로딩을 진행하여 체감 로딩 시간을 줄여야 한다. 특히 5장의 드래곤 비스트 대량 출현 이벤트는 오브젝트 풀링으로 처리한다.
>
> **[기획자 결정]** 수용. 컷씬은 챕터당 2~3개 핵심 장면만 제작, 홀로그램 브리핑 형식으로 통일. 전체 스킵 + 개별 스킵 옵션 모두 제공. 컷씬 중 백그라운드 로딩 적용.

## 2. 미션 보상 체계

### 2.1 보상 종류

| 보상 | 메인 미션 | 사이드 | 일일 | 주간 |
|------|-----------|--------|------|------|
| 경험치 | ◎ | ○ | ○ | ◎ |
| 크레딧 | ○ | ○ | ◎ | ○ |
| 장비 | ◎ (고정) | △ (랜덤) | × | △ |
| 스킬 포인트 | △ (일부) | × | × | × |
| 칭호 | △ (일부) | △ | × | × |
| 드래곤 코어 | △ (후반) | × | × | △ |

### 2.2 미션 난이도 보상 보정

- 권장 레벨보다 5 이상 높으면: 경험치 보상 -50%
- 권장 레벨보다 5 이상 낮으면: 표기 없이 보상 유지 (도전 보상)

> **[프로그래머 의견]** 보상 보정 공식은 서버 측에서만 계산하고, 클라이언트에는 최종 보상 수치만 전송해야 한다. 보정 공식이 클라이언트에 노출되면 최적 레벨 구간 계산 봇이 등장할 수 있다.
>
> **[기획자 결정]** 수용. 보상 계산은 서버 전용 로직으로 처리.

## 3. 일일/주간 미션 상세

### 3.1 일일 미션 풀 (랜덤 5개 배정)

| 카테고리 | 예시 | 보상 |
|----------|------|------|
| 소탕 | "XX 구역 호스틸 100기 격파" | 경험치 + 크레딧 |
| 채집 | "광물/에너지 셀 30개 채집" | 제작 재료 + 크레딧 |
| 작전구역 | "XX 인스턴스 1회 클리어" | 경험치 + 장비 컨테이너 |
| 제작 | "무기/장갑 5개 제작" | 제작 숙련도 + 크레딧 |
| PvP | "전장 1회 참여" | 명예 포인트 + 크레딧 |

### 3.2 주간 미션

| 미션 | 조건 | 보상 |
|------|------|------|
| 주간 보스 토벌 | 필드 보스 3회 격파 | 영웅 장비 컨테이너 |
| 주간 PvP | 전장 10회 참여 | 명예 포인트 대량 |
| 군단 활동 | 군단 작전구역 클리어 | 군단 자금 + 개인 보상 |

> **[디자이너 의견]** 일일 미션은 '의무감'이 아닌 '자연스러운 플레이 보상'이 되어야 한다. 소탕 100기는 30분 내외, 채집 30개는 20분 내외로 완료 가능한 수치를 유지하여 부담을 최소화한다.
>
> **[기획자 결정]** 수용. 일일 미션 전체 클리어 목표 시간: 60~90분. 일일 미션 전부 클리어 시 추가 완료 보너스(크레딧 + 드래곤 코어 조각) 지급.

## 4. 업적 시스템

### 4.1 업적 카테고리

| 카테고리 | 예시 | 보상 유형 |
|----------|------|-----------|
| 전투 | "호스틸 10,000기 격파", "보스 첫 격파" | 칭호, 스탯 보너스 |
| 탐사 | "모든 구역 방문", "숨겨진 구역 발견" | 칭호, 외형 |
| 제작 | "전설 장비 제작 성공" | 칭호, 제작 보너스 |
| PvP | "PvP 100승", "기지 쟁탈전 승리" | 칭호, PvP 장비 |
| 사회 | "군단 창설", "동맹 체결" | 칭호 |
| 수집 | "전 직업 2차 전환 달성", "전 종족 Lv.50" | 계정 보너스 |

### 4.2 업적 포인트 시스템

- 모든 업적 달성 시 **업적 포인트** 획득
- 업적 포인트 누적에 따른 단계별 보상:

| 누적 포인트 | 보상 |
|-------------|------|
| 100 | 칭호 "신참 용병" + 메디키트 상자 |
| 500 | 칭호 "베테랑 오퍼레이터" + 영웅 장비 선택권 |
| 1,000 | 칭호 "전설의 에이스" + 호버바이크 외형 |
| 2,000 | 칭호 "드래곤 헌터" + 전설 드론 선택권 |
| 5,000 | 칭호 "행성의 수호자" + 전용 홀로그램 이펙트 |

> **[기획자 의견]** 업적은 '자연스럽게 달성되는 것'과 '의도적으로 도전하는 것'의 비율을 7:3으로 유지한다. 플레이하다 보면 달성되는 업적이 많아야 성취감이 지속되고, 소수의 도전 업적이 하드코어 유저의 장기 목표가 된다.
>
> **[프로그래머 의견]** 업적 데이터는 서버에서 비동기로 체크하여 클라이언트 성능에 영향을 주지 않아야 한다. 특히 "호스틸 10,000기 격파" 같은 누적형 업적은 매 처치마다 DB 쿼리를 하지 않고, 인메모리 카운터를 사용한 뒤 주기적으로 동기화하는 방식이 효율적이다.
>
> **[기획자 결정]** 수용. 업적 체크는 서버 비동기 처리, 누적형은 인메모리 카운터 사용. 업적 달성 시 화면 중앙에 홀로그램 알림 연출.

## 5. 미션 UI/UX

### 5.1 미션 추적

| 기능 | 설명 |
|------|------|
| 미션 트래커 | 화면 우측에 최대 5개 미션 동시 추적 |
| 내비게이션 | 미션 목표 위치까지 홀로그램 경로 안내 |
| 자동 수락 | 일일/주간 미션은 로그인 시 자동 배정 |
| 미션 로그 | 완료/실패 미션 이력 열람 가능 |

### 5.2 미션 공유

- 파티 미션: 파티원 전원이 동일 미션 진행 시 진행도 공유
- 군단 미션: 군단원 기여도에 따라 보상 차등 분배
- 사이드 미션: 개인 진행 (공유 불가)

> **[디자이너 의견]** 홀로그램 내비게이션은 SF 세계관에 자연스럽게 녹아들면서도 직관적이어야 한다. 바닥에 투사되는 AR 경로선 스타일을 기본으로 하되, HUD 미니맵의 방향 표시와 연동하여 이중 가이드를 제공한다.
>
> **[기획자 결정]** 수용. 내비게이션은 AR 바닥 경로선(기본) + 미니맵 방향 표시(보조). 설정에서 경로선 ON/OFF 가능.
