# 군단 시스템 (Corps System)

- **프로젝트명**: Dragon
- **장르**: SF 리니지 라이크 MMORPG
- **버전**: 2.0
- **최종 수정일**: 2026-03-23

---

## 1. 군단 기본

### 1.1 군단 창설

| 항목 | 조건 |
|------|------|
| 창설 레벨 | Lv.20 이상 |
| 창설 비용 | 100,000 크레딧 |
| 최소 창단 인원 | 5명 (서명) |
| 군단명 | 2~16자, 중복 불가 |
| 군단 마크 | 16×16 커스텀 엠블럼 업로드 |

### 1.2 군단 등급

| 등급 | 최대 인원 | 필요 경험치 | 해금 기능 |
|------|-----------|------------|-----------|
| Lv.1 | 20명 | 창설 시 | 기본 군단 채팅, 군단 보급창고(50칸) |
| Lv.2 | 30명 | 50,000 | 군단 공지, 군단 스킬 1종 |
| Lv.3 | 40명 | 200,000 | 군단 보급창고 확장(100칸), 군단 스킬 3종 |
| Lv.4 | 60명 | 500,000 | 영토전 참여 자격, 군단 스킬 5종 |
| Lv.5 | 80명 | 1,000,000 | 기지 쟁탈전 참여 자격, 연합 체결(최대 2군단) |
| Lv.6 | 100명 | 3,000,000 | 군단 사령부 커스터마이징, 군단 스킬 전체 해금 |

- 군단 경험치: 군단원의 소탕/퀘스트/PvP 활동에서 자동 축적

### 1.3 군단 계급

| 계급 | 인원 | 권한 |
|------|------|------|
| **사령관** | 1명 | 모든 권한, 군단 해산 |
| **부사령관** | 최대 2명 | 가입 승인, 추방, 보급창고 관리 |
| **선임 대원** | 최대 5명 | 가입 승인, 군단 공지 |
| **정규 대원** | 제한 없음 | 군단 채팅, 보급창고 입출금 |
| **견습 대원** | 제한 없음 | 군단 채팅, 보급창고 열람만 |

> **[프로그래머 의견]** 군단 보급창고 접근 권한은 계급별로 세분화해야 한다. 특히 물자 인출 권한은 사령관/부사령관만 가능하거나, 일일 인출 한도를 설정할 수 있어야 군단 내부 도난을 방지할 수 있다.
>
> **[기획자 결정]** 수용. 군단 보급창고는 계급별 인출 권한 설정 + 일일 인출 한도(사령관이 설정) + 인출 로그 기록. 사령관은 모든 로그를 조회 가능.

## 2. 군단 스킬

군단 등급 상승에 따라 해금되며, 군단원 전원에 패시브로 적용.

| 스킬 | 해금 등급 | 효과 |
|------|----------|------|
| 군단의 결속 | Lv.2 | 군단원 근처(50m) 시 전 스탯 +2% |
| 전투 고양 | Lv.3 | 물리 공격력 +3% |
| 에너지 공명 | Lv.3 | 에너지 공격력 +3% |
| 방어막 강화 | Lv.3 | 물리/에너지 방어력 +3% |
| 기동 부스터 | Lv.4 | 이동 속도 +5% |
| 전사의 나노 강화 | Lv.4 | 최대 HP/EP +5% |
| 전장의 지배자 | Lv.5 | PvP 데미지 +3% |
| 드래곤의 가호 | Lv.6 | 드래곤 게이지 충전 속도 +10% |
| 군단의 재기동 | Lv.6 | 군단원 사망 시 50% 확률 즉시 부활(HP 30%) |

> **[기획자 의견]** 군단 스킬이 개인 전투력에 미치는 영향은 5~10% 수준으로 제한한다. 군단 소속의 이점은 분명하되, 솔로 플레이어와의 격차가 압도적이지 않아야 한다.

## 3. 연합 시스템

### 3.1 연합 구성

- 최대 **3개 군단**이 연합 결성 가능 (군단 Lv.5 이상)
- 연합 결성: 양 사령관 합의
- 연합 채팅 채널 공유
- 연합 군단원 간 PK 불가

### 3.2 교전 선언

- 군단 간 **교전 선언** 가능 (일방적)
- 교전 선언 시: 상대 군단원 공격 시 성향치 감소 없음
- 교전 해제: 선언한 군단이 해제하거나, 7일 자동 해제
- 쌍방 교전 시 **군단전** 성립: 킬 스코어 집계, 주간 정산 보상

> **[프로그래머 의견]** 교전 선언/해제의 상태 관리가 다수 군단 간에 발생하면 O(N²) 체크가 필요하다. 비트 플래그 기반 교전 테이블을 서버에 캐싱하여 PvP 판정 시 O(1) 조회가 되도록 최적화해야 한다.
>
> **[기획자 결정]** 수용. 서버 부하를 고려하여 동시 교전 선언 가능 수를 군단당 최대 5개로 제한한다.

## 4. 군단 콘텐츠

### 4.1 군단 퀘스트

- 주간 군단 퀘스트: 군단 경험치 + 군단 자금 보상
- 예시: "군단원 5명 이상 파티로 던전 클리어", "군단 작전 구역에서 적 개체 10,000기 소탕"

### 4.2 군단 던전

- 군단 전용 인스턴스 던전
- 주 1회 입장, 군단 Lv.3 이상
- 클리어 보상: 군단 자금 + 개인 보상 + 보급창고 보상

### 4.3 군단 랭킹

| 랭킹 종류 | 기준 | 보상 |
|-----------|------|------|
| 종합 랭킹 | 군단 레벨 + 영토 수 + 전투 점수 | 서버 1위 군단 전용 칭호 |
| PvP 랭킹 | 군단전 승률 + 킬 수 | PvP 보너스 |
| 활동 랭킹 | 군단원 평균 접속/활동 시간 | 군단 경험치 보너스 |

## 5. 군단 해산 및 탈퇴

| 항목 | 규칙 |
|------|------|
| 탈퇴 | 자유 탈퇴, 24시간 재가입 제한 |
| 추방 | 사령관/부사령관만 가능, 48시간 재가입 제한 |
| 해산 | 사령관만 가능, 7일 유예 기간 (군단원 공지) |
| 사령관 위임 | 사령관이 부사령관에게 위임 가능 |
| 사령관 장기 미접속 | 30일 미접속 시 부사령관에 자동 위임 |

> **[기획자 의견]** 사령관 장기 미접속 자동 위임은 군단이 죽지 않도록 하는 안전장치다. 많은 MMORPG에서 길드장 이탈로 길드 전체가 와해되는 사례가 있으므로, 이 시스템은 반드시 필요하다.
