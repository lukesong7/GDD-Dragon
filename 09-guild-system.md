# 혈맹 시스템 (Guild System)

- **프로젝트명**: Dragon
- **버전**: 1.0
- **최종 수정일**: 2026-03-23

---

## 1. 혈맹 기본

### 1.1 혈맹 생성

| 항목 | 조건 |
|------|------|
| 생성 레벨 | Lv.20 이상 |
| 생성 비용 | 100,000 골드 |
| 최소 창단 인원 | 5명 (서명) |
| 혈맹명 | 2~16자, 중복 불가 |
| 혈맹 마크 | 16×16 커스텀 엠블럼 업로드 |

### 1.2 혈맹 등급

| 등급 | 최대 인원 | 필요 경험치 | 해금 기능 |
|------|-----------|------------|-----------|
| Lv.1 | 20명 | 생성 시 | 기본 혈맹 채팅, 혈맹 창고(50칸) |
| Lv.2 | 30명 | 50,000 | 혈맹 공지, 혈맹 스킬 1종 |
| Lv.3 | 40명 | 200,000 | 혈맹 창고 확장(100칸), 혈맹 스킬 3종 |
| Lv.4 | 60명 | 500,000 | 영토전 참여 자격, 혈맹 스킬 5종 |
| Lv.5 | 80명 | 1,000,000 | 공성전 참여 자격, 동맹 체결(최대 2혈맹) |
| Lv.6 | 100명 | 3,000,000 | 혈맹 홀 커스터마이징, 혈맹 스킬 전체 해금 |

- 혈맹 경험치: 혈맹원의 사냥/퀘스트/PvP 활동에서 자동 축적

### 1.3 혈맹 직위

| 직위 | 인원 | 권한 |
|------|------|------|
| **혈맹주** | 1명 | 모든 권한, 혈맹 해산 |
| **부맹주** | 최대 2명 | 가입 승인, 추방, 창고 관리 |
| **수호기사** | 최대 5명 | 가입 승인, 혈맹 공지 |
| **기사** | 제한 없음 | 혈맹 채팅, 혈맹 창고 입출금 |
| **견습기사** | 제한 없음 | 혈맹 채팅, 혈맹 창고 열람만 |

> **[프로그래머 의견]** 혈맹 창고 접근 권한은 직위별로 세분화해야 한다. 특히 아이템 인출 권한은 혈맹주/부맹주만 가능하거나, 일일 인출 한도를 설정할 수 있어야 혈맹 내부 도난을 방지할 수 있다.
>
> **[기획자 결정]** 수용. 혈맹 창고는 직위별 인출 권한 설정 + 일일 인출 한도(혈맹주가 설정) + 인출 로그 기록. 혈맹주는 모든 로그를 조회 가능.

## 2. 혈맹 스킬

혈맹 등급 상승에 따라 해금되며, 혈맹원 전원에 패시브로 적용.

| 스킬 | 해금 등급 | 효과 |
|------|----------|------|
| 혈맹의 결속 | Lv.2 | 혈맹원 근처(50m) 시 전 스탯 +2% |
| 전투의 함성 | Lv.3 | 물리 공격력 +3% |
| 마력 공명 | Lv.3 | 마법 공격력 +3% |
| 수호의 방벽 | Lv.3 | 물리/마법 방어력 +3% |
| 신속의 깃발 | Lv.4 | 이동 속도 +5% |
| 용사의 기운 | Lv.4 | 최대 HP/MP +5% |
| 전장의 지배자 | Lv.5 | PvP 데미지 +3% |
| 용의 가호 | Lv.6 | 드래곤 게이지 충전 속도 +10% |
| 혈맹의 부활 | Lv.6 | 혈맹원 사망 시 50% 확률 즉시 부활(HP 30%) |

> **[기획자 의견]** 혈맹 스킬이 개인 전투력에 미치는 영향은 5~10% 수준으로 제한한다. 혈맹 소속의 이점은 분명하되, 솔로 플레이어와의 격차가 압도적이지 않아야 한다.

## 3. 동맹 시스템

### 3.1 동맹 구성

- 최대 **3개 혈맹**이 동맹 결성 가능 (혈맹 Lv.5 이상)
- 동맹 결성: 양 혈맹주 합의
- 동맹 채팅 채널 공유
- 동맹 혈맹원 간 PK 불가

### 3.2 적대 선언

- 혈맹 간 **적대 선언** 가능 (일방적)
- 적대 선언 시: 상대 혈맹원 공격 시 성향치 감소 없음
- 적대 해제: 선언한 혈맹이 해제하거나, 7일 자동 해제
- 쌍방 적대 시 **혈맹전** 성립: 킬 스코어 집계, 주간 정산 보상

> **[프로그래머 의견]** 적대 선언/해제의 상태 관리가 다수 혈맹 간에 발생하면 O(N²) 체크가 필요하다. 비트 플래그 기반 적대 테이블을 서버에 캐싱하여 PvP 판정 시 O(1) 조회가 되도록 최적화해야 한다.
>
> **[기획자 결정]** 수용. 서버 부하를 고려하여 동시 적대 선언 가능 수를 혈맹당 최대 5개로 제한한다.

## 4. 혈맹 콘텐츠

### 4.1 혈맹 퀘스트

- 주간 혈맹 퀘스트: 혈맹 경험치 + 혈맹 자금 보상
- 예시: "혈맹원 5명 이상 파티로 던전 클리어", "혈맹 사냥터에서 몬스터 10,000마리 처치"

### 4.2 혈맹 던전

- 혈맹 전용 인스턴스 던전
- 주 1회 입장, 혈맹 Lv.3 이상
- 클리어 보상: 혈맹 자금 + 개인 보상 + 혈맹 창고 보상

### 4.3 혈맹 랭킹

| 랭킹 종류 | 기준 | 보상 |
|-----------|------|------|
| 종합 랭킹 | 혈맹 레벨 + 영토 수 + 전투 점수 | 서버 1위 혈맹 전용 칭호 |
| PvP 랭킹 | 혈맹전 승률 + 킬 수 | PvP 보너스 |
| 활동 랭킹 | 혈맹원 평균 접속/활동 시간 | 혈맹 경험치 보너스 |

## 5. 혈맹 해산 및 탈퇴

| 항목 | 규칙 |
|------|------|
| 탈퇴 | 자유 탈퇴, 24시간 재가입 제한 |
| 추방 | 혈맹주/부맹주만 가능, 48시간 재가입 제한 |
| 해산 | 혈맹주만 가능, 7일 유예 기간 (혈맹원 공지) |
| 혈맹주 위임 | 혈맹주가 부맹주에게 위임 가능 |
| 혈맹주 장기 미접속 | 30일 미접속 시 부맹주에 자동 위임 |

> **[기획자 의견]** 혈맹주 장기 미접속 자동 위임은 혈맹이 죽지 않도록 하는 안전장치다. 많은 MMORPG에서 혈맹주 이탈로 혈맹 전체가 와해되는 사례가 있으므로, 이 시스템은 반드시 필요하다.
