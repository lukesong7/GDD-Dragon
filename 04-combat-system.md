## 1. 전투 기본 구조

### 1.1 타겟팅

| 방식 | 적용 대상 | 조작 |
|------|-----------|------|
| **클릭 타겟팅** | 일반 공격, 단일 대상 스킬 | 대상 클릭 또는 Tab 키 전환 |
| **논타겟팅(범위)** | 범위 스킬, 설치형 스킬 (궤도 폭격, 그레네이드 등) | 지점 지정 또는 자기 중심 |
| **자동 타겟팅** | 모바일 전용 | AI 기반 최적 대상 선택 |

- Tab 키: 가장 가까운 적 → 두 번째 가까운 적 순서로 전환
- PvP 상황에서는 적대적 플레이어 우선 타겟팅 옵션 제공

### 1.2 전투 템포

| 항목 | 수치 | 비고 |
|------|------|------|
| **GCD (글로벌 쿨다운)** | 1.5초 | 모든 스킬 공유 |
| **일반 공격 속도** | 무기별 상이 (1.0~2.5초) | DEX로 단축 가능 |
| **스킬 시전** | 즉시~3초 | 스킬별 상이, 피격 시 시전 지연 |
| **스킬 쿨다운** | 개별 3~120초 | GCD와 별도 |

> **[디자이너 의견]** GCD 1.5초는 리니지 감성의 묵직한 템포를 유지한다. 다만 에이전트 계열은 빠른 연타가 핵심인데, GCD 안에서 다중 타격 처리가 필요하다. 한 번 스킬 사용 시 플라즈마 나이프가 여러 차례 타격하는 이펙트가 나가는 식으로 '체감 속도'를 높여야 한다.
>
> **[프로그래머 의견]** 다중 타격 스킬은 서버 측에서 1회 판정하되, 클라이언트에서 다수 이펙트를 시간차 재생하는 방식이 서버 부하와 타격감 양쪽을 만족시킨다.
>
> **[기획자 결정]** 수용. 에이전트 계열 스킬은 GCD 1.5초 내에 2~5히트가 클라이언트에서 순차 표현되며, 서버는 총 데미지를 1회 판정한다.

## 2. 데미지 공식

### 2.1 기본 데미지 계산

```
최종 데미지 = (기본 데미지 - 방어력 감소) × 속성 보정 × 크리티컬 배율 × 랜덤 편차

기본 데미지 = 공격력 × 스킬 배율
방어력 감소 = 장갑수치 / (장갑수치 + 상수K)    ← K = 500 (비율 기반, 장갑 무한 스택 방지)
속성 보정 = 유리 속성 1.3 / 불리 속성 0.7 / 동일 속성 1.0
크리티컬 배율 = 크리티컬 시 1.5~2.5 (장비/스킬에 따라)
랜덤 편차 = 0.95~1.05 (±5%)
```

> **[프로그래머 의견]** 장갑 감소를 `장갑수치 / (장갑수치 + K)` 비율 기반으로 한 것은 좋다. 장갑이 아무리 높아도 데미지를 0으로 만들 수 없어 밸런스 붕괴를 방지한다. K값(500)은 밸런스 테이블에서 미세 조정 필요.
>
> **[기획자 결정]** K값은 서버 파라미터로 관리하여 핫픽스 없이 조정 가능하게 한다.

### 2.2 에너지/테크 공격 데미지 계산

```
최종 데미지 = (테크 공격력 × 스킬 배율 - 에너지 저항 감소) × 속성 보정 × 크리티컬 배율 × 랜덤 편차

에너지 저항 감소 = 에너지 저항 / (에너지 저항 + 상수K2)    ← K2 = 450
```

- 에너지/테크 공격은 물리 장갑을 무시하고 에너지 저항으로 별도 감쇠
- 테크노서, 메딕 등 EP 기반 클래스의 주력 공격 체계

### 2.3 PvP 데미지 보정

```
PvP 데미지 = PvE 데미지 × PvP 감쇠 계수 × 장비 PvP 보정

PvP 감쇠 계수 = 0.5 (PvE 대비 50% 데미지)
```

- PvP와 PvE 데미지를 분리하여 밸런싱
- 장비에 'PvP 공격력/방어력' 옵션 별도 존재

> **[기획자 의견]** PvP 감쇠 0.5는 시작점이다. 클래스별 PvP 추가 보정이 필요하며 이는 스킬 시스템 문서에서 구체화한다.

## 3. 속성 시스템

### 3.1 속성 분류

| 속성 | 시각 표현 | 상성 우위 | 상성 열위 |
|------|-----------|-----------|-----------|
| **플라즈마 (열)** | 주황-적색 발광 | 냉각 속성에 +30% | 전기 속성에 -20% |
| **전기 (전자)** | 청-백색 방전 | 플라즈마 속성에 +30% | 냉각 속성에 -20% |
| **냉각 (냉)** | 청-보라색 결정 | 전기 속성에 +30% | 플라즈마 속성에 -20% |

- 3속성 순환 상성(플라즈마 > 냉각 > 전기 > 플라즈마) + 중력(범용 보너스) + 무속성
- 색맹 대응을 위해 이펙트 형태도 차별화 (입자, 방전, 결정, 왜곡)

## 4. 상태이상 시스템

### 4.1 상태이상 분류

| 분류 | 상태이상 | 효과 | 최대 지속 |
|------|----------|------|-----------|
| **행동 제한** | 시스템 과부하(Stun) | 모든 행동 불가 | 3초 |
| | 중력파 속박(Root) | 이동 불가, 공격/스킬 가능 | 5초 |
| | 전자기 교란(Silence) | 스킬 사용 불가, 이동/일반 공격 가능 | 5초 |
| | EMP(Sleep) | 모든 행동 불가, 피격 시 해제 | 8초 |
| | 해킹(Fear) | 랜덤 방향 이동, 조작 불가 | 3초 |
| **능력 저하** | 구동 제한(Slow) | 이동 속도 -30~50% | 10초 |
| | 동력 저하(Weaken) | 공격력 -20~40% | 10초 |
| | 장갑 약화(Armor Break) | 물리 방어력 -20~40% | 10초 |
| **지속 피해** | 출혈(Bleed) | 2초마다 물리 데미지 | 10초 |

| **특수** | 냉각제 유출(Freeze) | 행동 불가 + 받는 피해 +20% | 2초 |
| | 회로 정지(Petrify) | 행동 불가 + 받는 피해 -50% | 4초 |

### 4.2 상태이상 저항 및 면역

- **저항**: WIS 기반. 상태이상 적용 시 `저항 확률 = WIS × 0.3%` 판정
- **면역**: 같은 CC(행동 제한)에 연속 피격 시 **3초간 해당 CC 면역** (DR 시스템)
- **정화**: 메딕 스킬 '시스템 정화', 나노 해독제 등으로 상태이상 제거

> **[프로그래머 의견]** CC 면역(DR) 시스템이 PvP 밸런스의 핵심이다. 같은 CC '종류'가 아닌 같은 CC '카테고리'(행동 제한 전체)에 면역을 적용해야 시스템 과부하→중력파 속박→해킹 순서 CC 체인을 방지할 수 있다.
>
> **[기획자 결정]** 행동 제한 카테고리 전체에 대해 DR을 적용한다. 첫 CC 풀 지속 → 2회 연속 CC 50% 지속 → 3회 연속 CC 면역(5초). 이후 DR 리셋.

## 5. 전투 보조 시스템

### 5.1 메디키트 및 소모품

| 아이템 | 효과 | 쿨다운 | 비고 |
|--------|------|--------|------|
| HP 메디키트 (소/중/대) | HP 즉시 회복 300/700/1500 | 10초 | 전투 중 사용 가능 |
| 배터리팩 (소/중/대) | EP 즉시 회복 200/500/1000 | 10초 | 전투 중 사용 가능 |
| 나노 해독제 | 나노 부식/상태이상 해제 | 30초 | |
| EMP 차폐기 | 냉각제 유출/구동 제한 해제 | 30초 | |
| 부스터 칩 | 30분간 공격력 +10% | 재사용 불가 | |
| 드래곤 코어 용액 | DG 즉시 +50% | 60초 | 드래곤 계열 전용 |

- 메디키트/배터리팩은 일반 공격/스킬 GCD와 별도 쿨다운
- PvP 중 소모품 연속 사용 제한: 30초 내 최대 3회

> **[기획자 의견]** 메디키트 무한 연타는 PvP를 '소모품전'으로 만든다. 30초 3회 제한은 자원 관리 전략을 추가하면서 극단적 소모품 의존을 방지한다.

### 5.2 버프/디버프 슬롯

- 동시 적용 가능 버프: **최대 12개**
- 동시 적용 가능 디버프: **최대 8개**
- 같은 효과의 버프는 중첩 불가, 높은 수치가 우선 적용
- 예외: 분대 버프와 자체 버프는 별도 슬롯으로 중첩 가능

## 6. 사망 시스템

### 6.1 사망 패널티

| 사망 원인 | 경험치 손실 | 장비 드롭 | 기타 |
|-----------|------------|-----------|------|
| PvE 사망 (호스틸) | 현재 레벨 경험치의 3% | 없음 | |
| PvP 사망 (정당방위) | 1% | 없음 | 공격 받은 쪽 |
| PvP 사망 (PK 피해자) | 0% | 없음 | PK 당한 쪽 보호 |
| PK 사망 (가해자 사망) | 5% | 5% 확률로 장비 1개 | 성향치 악화 |
| 기지 쟁탈전 사망 | 0% | 없음 | 쟁탈전 중 패널티 면제 |

### 6.2 부활 방식

| 방식 | 조건 | HP/EP 회복 | 위치 |
|------|------|-----------|------|
| 기지 부활 | 무조건 가능 | 50% / 50% | 가장 가까운 아군 기지 |
| 현장 부활 | 메딕 부활 스킬 '전장 소생' | 30% / 30% | 사망 위치 |
| 긴급 부활 모듈 | 소비 아이템 사용 | 70% / 70% | 사망 위치 |
| 즉시 부활 | 프리미엄 아이템 | 100% / 100% | 사망 위치 |

> **[프로그래머 의견]** 즉시 부활 프리미엄 아이템은 PvP에서 악용될 수 있다. 기지 쟁탈전과 PvP 상황에서는 즉시 부활 사용 불가 또는 쿨다운을 길게 가져가야 한다.
>
> **[디자이너 의견]** 사망 시 파워아머 시스템 과부하 연출이 중요하다. 단순 쓰러짐이 아닌, 아머 전원 차단 이펙트 + 화면 노이즈 + "리부트 선택" UI가 자연스럽게 전환되어야 한다.
>
> **[기획자 결정]** 즉시 부활은 PvP/기지 쟁탈전에서 사용 불가. 사망 연출은 1.5초 시스템 셧다운 이펙트 → 부활 선택 UI 표시로 처리한다.

## 7. 자동 전투

### 7.1 자동 전투 범위

| 기능 | PC | 모바일 |
|------|-----|--------|
| 자동 공격 (일반 공격 반복) | O | O |
| 자동 스킬 사용 | X | O |
| 자동 소모품 사용 | HP/EP 임계치 설정 | HP/EP 임계치 설정 |
| 자동 이동 (소탕 구역 순회) | X | O |
| 자동 아이템 수집 | 등급 필터 | 등급 필터 |

> **[기획자 의견]** PC에서 자동 전투를 제한하는 이유는 두 가지다. 첫째, PC 유저는 조작의 재미를 기대한다. 둘째, PC 자동 전투를 완전 개방하면 매크로/봇과 구분이 불가능해져 단속이 어려워진다. 모바일은 이동 중 플레이 특성상 자동 전투를 허용하되, PvP 상황 진입 시 자동 전투 해제를 권장한다.

### 7.2 자동 전투 제한

- 자동 전투 시 **획득 경험치 -20%** (수동 전투 인센티브)
- 자동 전투 연속 가동 시간: 기본 2시간, 프리미엄 패스 6시간
- PvP 지역에서 자동 전투 시 피격 알림 (모바일 푸시)
- 자동 전투 중 PK 공격 불가 (피격 시만 반격)

| 전설 (Legend) | 310 | 400 | 고정 옵션 2 + 랜덤 2개 + 세트 효과 |
| 신화 (Mythic) | 400 | 520 | 고정 옵션 3 + 특수 능력 + 외형 변환 |

> **[디자이너 의견]** SF 테마에서 무기 이펙트 차별화가 핵심이다. 에너지 블레이드는 청-백색 플라즈마 잔상, 라이플은 탄도 궤적 + 탄착 이펙트, 레일건은 관통 선 이펙트 + 대기 왜곡. 등급에 따라 이펙트 강도와 색상이 변해야 한다. 신화 등급은 무기 자체에서 상시 에너지 파동이 방출되는 아이들 이펙트를 추가한다.

## 10. PvP 전투

### 10.1 PvP 모드

| 모드 | 설명 |
|------|------|
| 평화 모드 | PvP 불가. 기본 모드 |
| 전투 모드 | 다른 플레이어 공격 가능. 수동 전환 |
| 군단전 모드 | 적대 군단원 자동 타겟 가능 |
| 전면전 모드 | 군단/아군 외 모든 플레이어가 적 |

### 10.2 성향 시스템

| 성향 | 범위 | 효과 |
|------|------|------|
| 준법 (Lawful) | +32,767 ~ +1 | NPC 상점 할인, 사망 패널티 감소 |
| 중립 (Neutral) | 0 | 기본 상태 |
| 무법 (Chaotic) | -1 ~ -32,768 | 기지 입장 제한, NPC 공격, 사망 시 장비 드롭률 증가 |

- 자신보다 레벨이 낮은 평화 모드 플레이어를 공격하면 성향 대폭 하락
- 성향 회복: 호스틸 소탕, 일일 임무 수행, 시간 경과

### 10.3 PvP 밸런스 조정

- PvP 시 스킬 데미지 별도 계수 적용 (PvE 대비 60~80%)
- 제어 계열 상태이상 지속시간 PvP 시 50% 감소
- 레벨 차이 보정: 5레벨 이상 차이 시 약자 측에 방어 보너스
- 장비 등급 격차 보정: 신화급 vs 일반급 시 데미지 상한 적용
