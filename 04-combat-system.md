- **최종 수정일**: 2026-03-23

## 1. 전투 기본 구조

### 1.1 타겟팅

| 방식 | 적용 대상 | 조작 |
|------|-----------|------|
| **클릭 타겟팅** | 일반 공격, 단일 대상 스킬 | 대상 클릭 또는 Tab 키 전환 |
| **논타겟팅(범위)** | 범위 스킬, 설치형 스킬 | 지점 지정 또는 자기 중심 |
| **자동 타겟팅** | 모바일 전용 | AI 기반 최적 대상 선택 |

- Tab 키: 가장 가까운 적 → 두 번째 가까운 적 순서로 전환
- PvP 상황에서는 적대적 플레이어 우선 타겟팅 옵션 제공

### 1.2 전투 템포

| 항목 | 수치 | 비고 |
|------|------|------|
| **GCD (글로벌 쿨다운)** | 1.5초 | 모든 스킬 공유 |
| **일반 공격 속도** | 무기별 상이 (1.0~2.5초) | DEX로 단축 가능 |
| **스킬 캐스팅** | 즉시~3초 | 스킬별 상이, 피격 시 캐스팅 지연 |
| **스킬 쿨다운** | 개별 3~120초 | GCD와 별도 |

> **[디자이너 의견]** GCD 1.5초는 리니지 감성에 맞는 묵직한 템포다. 다만 도적 계열은 빠른 연타가 핵심인데, GCD 안에서 다중 타격 처리가 필요하다. 한 번 스킬 사용 시 여러 타격 이펙트가 나가는 식으로 '체감 속도'를 높여야 한다.
>
> **[프로그래머 의견]** 다중 타격 스킬은 서버 측에서 1회 판정하되, 클라이언트에서 다수 이펙트를 시간차 재생하는 방식이 서버 부하와 타격감 양쪽을 만족시킨다.
>
> **[기획자 결정]** 수용. 도적 계열 스킬은 GCD 1.5초 내에 2~5히트가 클라이언트에서 순차 표현되며, 서버는 총 데미지를 1회 판정한다.

## 2. 데미지 공식

### 2.1 기본 데미지 계산

```
최종 데미지 = (기본 데미지 - 방어력 감소) × 속성 보정 × 크리티컬 배율 × 랜덤 편차

기본 데미지 = 공격력 × 스킬 배율
방어력 감소 = 방어력 / (방어력 + 상수K)    ← K = 500 (비율 기반, 방어력 무한 스택 방지)
속성 보정 = 유리 속성 1.3 / 불리 속성 0.7 / 동일 속성 1.0
크리티컬 배율 = 크리티컬 시 1.5~2.5 (장비/스킬에 따라)
랜덤 편차 = 0.95~1.05 (±5%)
```

> **[프로그래머 의견]** 방어력 감소를 `방어력 / (방어력 + K)` 비율 기반으로 한 것은 좋다. 방어력이 아무리 높아도 데미지를 0으로 만들 수 없어 밸런스 붕괴를 방지한다. K값(500)은 밸런스 테이블에서 미세 조정 필요.
>
> **[기획자 결정]** K값은 서버 파라미터로 관리하여 핫픽스 없이 조정 가능하게 한다.

### 2.2 PvP 데미지 보정

```
PvP 데미지 = PvE 데미지 × PvP 감쇠 계수 × 장비 PvP 보정

PvP 감쇠 계수 = 0.5 (PvE 대비 50% 데미지)
```

- PvP와 PvE 데미지를 분리하여 밸런싱
- 장비에 'PvP 공격력/방어력' 옵션 별도 존재

> **[기획자 의견]** PvP 감쇠 0.5는 시작점이다. 직업별 PvP 추가 보정이 필요하며 이는 스킬 시스템 문서에서 구체화한다.

## 3. 상태이상 시스템

### 3.1 상태이상 분류

| 분류 | 상태이상 | 효과 | 최대 지속 |
|------|----------|------|-----------|
| **행동 제한** | 기절(Stun) | 모든 행동 불가 | 3초 |
| | 속박(Root) | 이동 불가, 공격/스킬 가능 | 5초 |
| | 침묵(Silence) | 스킬 사용 불가, 이동/일반 공격 가능 | 5초 |
| | 수면(Sleep) | 모든 행동 불가, 피격 시 해제 | 8초 |
| | 공포(Fear) | 랜덤 방향 이동, 조작 불가 | 3초 |
| **능력 저하** | 둔화(Slow) | 이동 속도 -30~50% | 10초 |
| | 약화(Weaken) | 공격력 -20~40% | 10초 |
| | 파쇄(Armor Break) | 방어력 -20~40% | 10초 |
| **지속 피해** | 출혈(Bleed) | 2초마다 물리 데미지 | 10초 |
| | 화상(Burn) | 2초마다 마법 데미지 | 8초 |
| | 중독(Poison) | 3초마다 고정 데미지 + 힐량 -50% | 15초 |
| **특수** | 빙결(Freeze) | 기절 + 받는 피해 +20% | 2초 |
| | 석화(Petrify) | 기절 + 받는 피해 -50% | 4초 |

### 3.2 상태이상 저항 및 면역

- **저항**: WIS 기반. 상태이상 적용 시 `저항 확률 = WIS × 0.3%` 판정
- **면역**: 같은 CC(행동 제한)에 연속 피격 시 **3초간 해당 CC 면역** (DR 시스템)
- **정화**: 힐러 스킬, 해독제 등으로 상태이상 제거

> **[프로그래머 의견]** CC 면역(DR) 시스템이 PvP 밸런스의 핵심이다. 같은 CC '종류'가 아닌 같은 CC '카테고리'(행동 제한 전체)에 면역을 적용해야 스턴→속박→공포 순서 CC 체인을 방지할 수 있다.
>
> **[기획자 결정]** 행동 제한 카테고리 전체에 대해 DR을 적용한다. 첫 CC 풀 지속 → 2회 연속 CC 50% 지속 → 3회 연속 CC 면역(5초). 이후 DR 리셋.

## 4. 전투 보조 시스템

### 4.1 물약 및 소모품

| 아이템 | 효과 | 쿨다운 | 비고 |
|--------|------|--------|------|
| HP 물약 (소/중/대) | HP 즉시 회복 300/700/1500 | 10초 | 전투 중 사용 가능 |
| MP 물약 (소/중/대) | MP 즉시 회복 200/500/1000 | 10초 | 전투 중 사용 가능 |
| 해독제 | 중독 해제 | 30초 | |
| 해동제 | 빙결/둔화 해제 | 30초 | |
| 축복 주문서 | 30분간 공격력 +10% | 재사용 불가 | |
| 용의 피약 | DG 즉시 +50% | 60초 | 드래곤 계열 전용 |

- 물약은 일반 공격/스킬 GCD와 별도 쿨다운
- PvP 중 물약 연속 사용 제한: 30초 내 최대 3회

> **[기획자 의견]** 물약 무한 연타는 PvP를 '물약전'으로 만든다. 30초 3회 제한은 자원 관리 전략을 추가하면서 극단적 물약 의존을 방지한다.

### 4.2 버프/디버프 슬롯

- 동시 적용 가능 버프: **최대 12개**
- 동시 적용 가능 디버프: **최대 8개**
- 같은 효과의 버프는 중첩 불가, 높은 수치가 우선 적용
- 예외: 파티 버프와 자체 버프는 별도 슬롯으로 중첩 가능

## 5. 사망 시스템

### 5.1 사망 패널티

| 사망 원인 | 경험치 손실 | 장비 드롭 | 기타 |
|-----------|------------|-----------|------|
| PvE 사망 (몬스터) | 현재 레벨 경험치의 3% | 없음 | |
| PvP 사망 (정당방위) | 1% | 없음 | 공격 받은 쪽 |
| PvP 사망 (PK 피해자) | 0% | 없음 | PK 당한 쪽 보호 |
| PK 사망 (가해자 사망) | 5% | 5% 확률로 장비 1개 | 성향치 악화 |
| 공성전 사망 | 0% | 없음 | 공성전 중 패널티 면제 |

### 5.2 부활 방식

| 방식 | 조건 | HP/MP 회복 | 위치 |
|------|------|-----------|------|
| 마을 부활 | 무조건 가능 | 50% / 50% | 가장 가까운 마을 |
| 현장 부활 | 힐러 부활 스킬 | 30% / 30% | 사망 위치 |
| 부활 주문서 | 소모품 사용 | 70% / 70% | 사망 위치 |
| 즉시 부활 | 캐시 아이템 | 100% / 100% | 사망 위치 |

> **[프로그래머 의견]** 즉시 부활 캐시 아이템은 PvP에서 악용될 수 있다. 공성전과 PvP 상황에서는 즉시 부활 사용 불가 또는 쿨다운을 길게 가져가야 한다.
>
> **[디자이너 의견]** 사망 시 화면 연출도 중요하다. 단순 쓰러짐이 아닌 잠깐의 그레이스케일 화면 + "부활 선택" UI가 자연스럽게 전환되어야 한다.
>
> **[기획자 결정]** 즉시 부활은 PvP/공성전에서 사용 불가. 사망 연출은 1.5초 페이드아웃 → 부활 선택 UI 표시로 처리한다.

## 6. 자동 전투

### 6.1 자동 전투 범위

| 기능 | PC | 모바일 |
|------|-----|--------|
| 자동 공격 (일반 공격 반복) | O | O |
| 자동 스킬 사용 | X | O |
| 자동 물약 사용 | HP/MP 임계치 설정 | HP/MP 임계치 설정 |
| 자동 이동 (사냥터 순회) | X | O |
| 자동 아이템 줍기 | 등급 필터 | 등급 필터 |

> **[기획자 의견]** PC에서 자동 전투를 제한하는 이유는 두 가지다. 첫째, PC 유저는 조작의 재미를 기대한다. 둘째, PC 자동 전투를 완전 개방하면 매크로/봇과 구분이 불가능해져 단속이 어려워진다. 모바일은 이동 중 플레이 특성상 자동 전투를 허용하되, PvP 상황 진입 시 자동 전투 해제를 권장한다.

### 6.2 자동 전투 제한

- 자동 전투 시 **획득 경험치 -20%** (수동 전투 인센티브)
- 자동 전투 연속 가동 시간: 기본 2시간, 프리미엄 패스 6시간
- PvP 지역에서 자동 전투 시 피격 알림 (모바일 푸시)
- 자동 전투 중 PK 공격 불가 (피격 시만 반격)
