# 수익화 모델 (Monetization)

- **프로젝트명**: Dragon
- **버전**: 1.0
- **최종 수정일**: 2026-03-23

---

## 1. 수익화 철학

### 1.1 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **No Pay to Win** | 캐시로 전투력을 직접 구매할 수 없음 |
| **외형/편의 중심** | 코스튬, 탈것 외형, 편의 기능이 매출의 주축 |
| **시간 단축 허용** | 경험치 부스트, 자동사냥 연장 등 '시간' 관련만 허용 |
| **희소성 설계** | 한정판/시즌 한정 외형으로 수집 욕구 자극 |
| **공정 경쟁** | 캐시 아이템이 PvP 밸런스를 훼손하지 않음 |

> **[기획자 의견]** P2W를 하면 단기 매출은 오르지만, 서버 수명이 단축된다. 리니지W의 사례에서 과금 피로가 서버 통합과 유저 이탈로 이어지는 것을 확인했다. Dragon은 장기 운영을 목표로 하므로 P2W를 철저히 배제한다.

## 2. 과금 상품

### 2.1 프리미엄 패스 (구독형)

| 항목 | 내용 |
|------|------|
| 가격 | 월 ₩15,000 |
| 경험치 부스트 | +30% |
| 자동사냥 시간 | 2시간 → 6시간 |
| 오프라인 상점 | 개설 가능 |
| 출석 보상 강화 | 출석 보상 2배 + 3일 복구 |
| 텔레포트 할인 | 50% 할인 |
| 인벤토리 | +20칸 |
| 경매장 등록 | 10개 → 20개 |

### 2.2 시즌 패스

| 항목 | 내용 |
|------|------|
| 가격 | ₩30,000 / 3개월 시즌 |
| 구조 | 무료 트랙 + 유료 트랙 |
| 유료 트랙 보상 | 시즌 한정 코스튬, 탈것 외형, 칭호, 이펙트 |
| 진행 방식 | 일일/주간 미션으로 시즌 포인트 획득 → 레벨업 |
| 최대 레벨 | 100 |

### 2.3 캐시샵 상품

| 카테고리 | 상품 예시 | 가격대 |
|----------|-----------|--------|
| **코스튬** | 전신 의상 세트 | ₩15,000~30,000 |
| **무기 외형** | 무기 스킨 (성능 무관) | ₩10,000~20,000 |
| **탈것 외형** | 탈것 스킨 | ₩15,000~30,000 |
| **이펙트** | 오라, 발자국, 이동 이펙트 | ₩5,000~15,000 |
| **편의** | 인벤토리 확장, 창고 확장 | ₩5,000~10,000 |
| **소모품** | 경험치 부스트(50%, 1시간), 리셋 아이템 | ₩1,000~5,000 |
| **강화 보조** | 축복 주문서, 보호 주문서 | ₩3,000~10,000 |

> **[프로그래머 의견]** 강화 보조 아이템을 캐시에서 판매하면 '간접 P2W' 논란이 있을 수 있다. 인게임 제작으로도 획득 가능하게 하되, 캐시 구매가 '시간 단축' 수준으로 느껴지도록 밸런싱해야 한다.
>
> **[기획자 결정]** 축복/보호 주문서는 인게임 제작으로도 획득 가능. 캐시 구매 수량 제한: 주간 5개까지. 이렇게 하면 소과금 유저도 접근 가능하면서 고과금 유저의 독점을 방지.

## 3. 가챠/랜덤박스 정책

### 3.1 기본 방침

- **확률 전부 공개**: 모든 랜덤박스의 확률을 게임 내 + 공식 사이트에 공개
- **천장 시스템**: 최대 N회 구매 시 최고 등급 보상 확정
- **외형 전용**: 랜덤박스에 전투력 아이템 포함 금지

### 3.2 코스튬 가챠 예시

| 등급 | 확률 | 천장 |
|------|------|------|
| 일반 코스튬 | 60% | - |
| 고급 코스튬 | 30% | - |
| 희귀 코스튬 | 8% | - |
| 전설 코스튬 | 2% | 50회 |

- 1회 비용: ₩3,000
- 중복 시: 코스튬 분해 토큰 제공 → 토큰으로 원하는 코스튬 교환

> **[디자이너 의견]** 코스튬 가챠가 매출의 핵심이 되려면 코스튬 디자인이 '반드시 갖고 싶은' 수준이어야 한다. 분기별 테마를 정하고(판타지, 사이버펑크, 동양풍 등), 각 테마의 전설 코스튬은 한정 수량 느낌을 주는 것이 효과적이다.
>
> **[기획자 결정]** 수용. 분기별 테마 코스튬 시즌 운영. 시즌 종료 후 해당 코스튬은 1년간 미판매 → 재등장 시 '복각' 이벤트로 운영.

## 4. 매출 예측 구조

### 4.1 유저 세그먼트별 예상 ARPPU

| 세그먼트 | 비율 | 월 ARPPU |
|----------|------|----------|
| 무과금 | 60% | ₩0 |
| 소과금 (패스만) | 25% | ₩15,000~30,000 |
| 중과금 | 12% | ₩50,000~100,000 |
| 고과금 (고래) | 3% | ₩200,000~500,000 |

### 4.2 매출 구성 목표

| 항목 | 비율 |
|------|------|
| 프리미엄 패스 | 30% |
| 시즌 패스 | 15% |
| 코스튬/외형 | 30% |
| 편의/소모품 | 15% |
| 기타 (강화보조 등) | 10% |

> **[기획자 의견]** 프리미엄 패스와 시즌 패스가 전체 매출의 45%를 차지하도록 설계한다. 안정적인 구독 기반 매출이 서비스 지속성의 핵심이다. 충동 구매(가챠)에 의존하면 매출 변동이 커 운영 계획이 불안정해진다.
