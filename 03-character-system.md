# 캐릭터 시스템 (Character System)

- **프로젝트명**: Dragon
- **버전**: 1.0
- **최종 수정일**: 2026-03-23

---

## 1. 스탯 시스템

### 1.1 기본 스탯 (6종)

| 스탯 | 약칭 | 효과 |
|------|------|------|
| **힘 (Strength)** | STR | 근접 물리 공격력, 장비 무게 한도 |
| **민첩 (Dexterity)** | DEX | 원거리 공격력, 명중률, 회피율, 공격 속도 |
| **체력 (Constitution)** | CON | 최대 HP, HP 자연 회복, 방어력 보정 |
| **지능 (Intelligence)** | INT | 마법 공격력, 최대 MP, 마법 크리티컬 |
| **정신 (Wisdom)** | WIS | MP 자연 회복, 마법 방어력, 버프 지속시간 |
| **운 (Luck)** | LUK | 크리티컬 확률, 드롭률 보정, 강화 성공률 미세 보정 |

### 1.2 스탯 포인트 배분

- 레벨업 시 **5포인트** 획득
- 자유 배분 방식 (리스탯 아이템으로 초기화 가능)
- 직업별 권장 스탯이 있으나 강제하지 않음

> **[프로그래머 의견]** LUK 스탯의 '강화 성공률 미세 보정'은 체감이 어려우면 유저 불만의 원인이 된다. 수치를 공개할 것인지, 히든으로 둘 것인지 결정 필요.
>
> **[기획자 결정]** LUK의 강화 성공률 보정은 **LUK 10당 0.1%** 수준으로 미미하게 유지하고, 툴팁에 명시한다. '체감은 어렵지만 없지는 않다' 수준으로, LUK 올인 빌드가 강화에서 유의미한 이점을 가지지 않도록 한다.

### 1.3 전투 스탯 (파생 스탯)

| 전투 스탯 | 산출 공식 (개략) |
|-----------|-----------------|
| 물리 공격력 | STR × 2 + 무기 공격력 |
| 마법 공격력 | INT × 2.5 + 무기 마법력 |
| 물리 방어력 | CON × 1.5 + 방어구 방어력 |
| 마법 방어력 | WIS × 1.5 + 방어구 마법 방어력 |
| 최대 HP | 기본 HP + (CON × 15) |
| 최대 MP | 기본 MP + (INT × 8) + (WIS × 5) |
| 명중률 | DEX × 0.5 + 레벨 보정 |
| 회피율 | DEX × 0.3 + 장비 보정 |
| 크리티컬 확률 | LUK × 0.2% + DEX × 0.1% + 장비 보정 |
| 크리티컬 데미지 | 기본 150% + STR/INT 보정 |
| 공격 속도 | 기본 무기 속도 + DEX 보정 |
| 이동 속도 | 기본 300 + 장비/버프 보정 |

> **[프로그래머 의견]** 공식이 단순한 선형이라 밸런스 조정이 용이하다. 다만 엔드게임에서 스탯 수치가 높아지면 물리/마법 공격력 차이가 과도하게 벌어질 수 있다. 레벨 보정 계수를 넣어 스케일링을 제어해야 한다.
>
> **[기획자 결정]** Lv.70 이상 구간에서 스탯 1포인트당 효율에 체감 보정(diminishing returns)을 적용한다. 구체적 수치는 밸런스 테스트에서 결정.

## 2. 직업 시스템

### 2.1 직업 계열 상세

#### 전사 계열 — 근접 물리 딜러/탱커

| 단계 | 직업 | 역할 | 주력 무기 | 핵심 특성 |
|------|------|------|-----------|-----------|
| 기본 | 파이터 | 만능 | 한손검+방패 | 기본 근접 공격 |
| 1차 전직 | 나이트 | 탱커 | 한손검+방패 | 높은 방어력, 도발, 파티 보호 |
| 1차 전직 | 워로드 | 딜러 | 양손 대검/창 | 범위 공격, 높은 단일 피해 |
| 2차 전직 | 다크나이트 | 탱커+ | 한손검+방패 | 어둠 속성 흡혈 탱킹 |
| 2차 전직 | 드래곤슬레이어 | 딜러+ | 양손 대검 | 드래곤 킬러 특수 스킬, 대보스 특화 |

#### 마법사 계열 — 원거리 마법 딜러

| 단계 | 직업 | 역할 | 주력 무기 | 핵심 특성 |
|------|------|------|-----------|-----------|
| 기본 | 메이지 | 마법 딜 | 스태프 | 기본 원소 마법 |
| 1차 전직 | 위자드 | 단일 딜 | 스태프 | 고데미지 단일 마법, 마나 효율 |
| 1차 전직 | 엘레멘탈리스트 | 범위 딜 | 오브 | 광역 원소 마법, CC기 |
| 2차 전직 | 아크메이지 | 극딜 | 스태프 | 극한의 마법 데미지, 글래스캐논 |
| 2차 전직 | 드래곤세이지 | 유틸 | 오브 | 드래곤 에센스 활용 마법, 속성 전환 |

#### 궁수 계열 — 원거리 물리 딜러

| 단계 | 직업 | 역할 | 주력 무기 | 핵심 특성 |
|------|------|------|-----------|-----------|
| 기본 | 레인저 | 원거리 딜 | 활 | 기본 원거리 공격 |
| 1차 전직 | 호크아이 | 저격 | 장궁 | 긴 사거리, 높은 크리티컬 |
| 1차 전직 | 트래커 | 스카웃 | 석궁 | 함정 설치, 은신 탐지, 이속 |
| 2차 전직 | 윈드워커 | 기동 딜 | 장궁 | 바람 속성 스킬, 최고 이동속도 |
| 2차 전직 | 드래곤아처 | 폭딜 | 용골 활 | 드래곤 화살(속성 화살), 관통 |

#### 도적 계열 — 근접 물리 딜러/어쌔신

| 단계 | 직업 | 역할 | 주력 무기 | 핵심 특성 |
|------|------|------|-----------|-----------|
| 기본 | 로그 | 근접 딜 | 단검 쌍수 | 빠른 공격, 은신 |
| 1차 전직 | 어쌔신 | 암살 | 단검 쌍수 | 은신→기습, 치명적 첫 타 |
| 1차 전직 | 팬텀 | 교란 | 단검+투척 | 환영 소환, 교란, 도주 |
| 2차 전직 | 블레이드댄서 | 연속 딜 | 쌍검 | 연쇄 공격 콤보, 최고 DPS |
| 2차 전직 | 드래곤섀도우 | 암살+ | 용아 단검 | 그림자 이동, 즉사기(조건부) |

#### 치유사 계열 — 힐러/서포터

| 단계 | 직업 | 역할 | 주력 무기 | 핵심 특성 |
|------|------|------|-----------|-----------|
| 기본 | 프리스트 | 힐러 | 메이스+성서 | 기본 힐, 버프 |
| 1차 전직 | 비숍 | 주힐러 | 메이스+성서 | 강력한 힐, 부활, 파티 버프 |
| 1차 전직 | 오라클 | 디버퍼 | 완드+주술서 | 적 디버프, 저주, 보조 딜 |
| 2차 전직 | 하이에로판트 | 대힐러 | 지팡이+성서 | 대규모 전투 힐 특화, 영역 버프 |
| 2차 전직 | 드래곤프리스트 | 하이브리드 | 용골 지팡이 | 드래곤 에센스로 힐+딜 전환 가능 |

> **[디자이너 의견]** 각 전직 경로 선택이 외형에도 명확히 반영되어야 한다. 나이트는 중갑+풀 플레이트, 워로드는 경갑+거대 무기. 같은 계열이라도 실루엣이 달라야 필드에서 즉시 구분 가능하다.
>
> **[프로그래머 의견]** 드래곤 계열 2차 전직은 공통적으로 드래곤 에센스 소모 스킬을 사용한다. 에센스를 별도 자원(게이지)으로 관리하는 시스템이 필요하다. 이 게이지는 전투 중 특정 행동으로 충전되고, 강력한 스킬에 소모되는 형태가 적합하다.
>
> **[기획자 결정]** 수용. **드래곤 게이지(DG)** 시스템을 도입한다. HP/MP 외에 드래곤 계열 전용 3번째 자원으로, 전투 시 자연 충전 + 드래곤 에센스 아이템으로 즉시 충전 가능. 비드래곤 계열 2차 전직도 DG를 갖지만 충전 속도와 활용도가 낮다.

### 2.2 전직 조건

| 전직 | 레벨 | 추가 조건 |
|------|------|-----------|
| 1차 전직 | Lv.30 | 전직 퀘스트 완료 (솔로 보스 처치) |
| 2차 전직 | Lv.60 | 전직 퀘스트 완료 (파티 던전) + 특수 아이템 수집 |

- 전직 후 되돌릴 수 없음
- 2차 전직에서 드래곤 계열 선택 시 드래곤 에센스 5개 필요

> **[기획자 의견]** 전직 불가역성은 캐릭터에 대한 애착을 높이고, 서버 내 직업 다양성을 유지하는 핵심 장치다. '전직 실수'에 대한 불만은 전직 전 체험 시스템(임시 전직 30분 체험)으로 완화한다.

## 3. 레벨링 시스템

### 3.1 경험치 테이블

| 구간 | 레벨업 소요 시간 (평균) | 누적 플레이 시간 | 콘텐츠 |
|------|------------------------|-----------------|--------|
| Lv.1~10 | 30분~1시간/레벨 | ~8시간 | 튜토리얼, 솔로 사냥 |
| Lv.11~30 | 1~2시간/레벨 | ~50시간 | 솔로→파티 전환기 |
| Lv.31~50 | 3~5시간/레벨 | ~150시간 | 파티 사냥 시작 |
| Lv.51~70 | 8~15시간/레벨 | ~400시간 | 본격 파티/혈맹 활동 |
| Lv.71~85 | 20~40시간/레벨 | ~800시간 | 엔드게임 진입 |
| Lv.86~99 | 50~100시간/레벨 | ~2000시간 | 최상위 목표 |

### 3.2 전생(리버스) 시스템

- Lv.99 달성 시 **전생** 가능
- 전생 시: 레벨 → Lv.1로 리셋, 전생 보너스 스탯 획득
- 전생 횟수에 따른 보너스:

| 전생 횟수 | 보너스 |
|-----------|--------|
| 1회 | 전 스탯 +5, 전생 칭호 |
| 2회 | 전 스탯 +10, 전용 스킬 1개 |
| 3회 | 전 스탯 +15, 전용 외형(용의 눈) |
| 4회+ | 전 스탯 +5 누적, 전용 외형 강화 |

> **[프로그래머 의견]** 전생 시 레벨이 1로 돌아가면 장비 착용 조건(레벨 제한)과 충돌한다. 전생 캐릭터는 레벨 제한을 무시하거나, '전생 레벨'이라는 별도 수치를 참조해야 한다.
>
> **[기획자 결정]** 전생 캐릭터는 이전 레벨에서 장착했던 장비를 그대로 유지한다. 전생 후 새로운 장비 착용 시에는 현재 레벨 기준을 따르되, '전생 횟수당 장비 레벨 제한 -10' 보정을 적용한다.

## 4. 외형 커스터마이징

### 4.1 캐릭터 생성 시 커스터마이징

| 항목 | 선택지 |
|------|--------|
| 성별 | 남/여 |
| 얼굴형 | 8종 (종족별) |
| 헤어스타일 | 15종 (종족별) |
| 헤어 컬러 | 자유 색상 선택 |
| 눈 모양 | 6종 |
| 눈 컬러 | 자유 색상 선택 |
| 피부톤 | 종족별 범위 내 자유 선택 |
| 체형 | 3단계 (마른/보통/건장) |
| 문신/흉터 | 10종 (선택) |

### 4.2 인게임 외형 변화

| 요소 | 조건 | 변화 |
|------|------|------|
| 장비 외형 | 장비 장착 시 | 장비에 따라 외형 변경 |
| 강화 이펙트 | +7 이상 장비 | 글로우(+7), 오라(+10), 특수효과(+15) |
| 드래곤 각성 | 각성 단계 | 용의 눈(2단계), 비늘(3단계), 날개(4단계), 뿔(5단계) |
| 칭호 | 업적/전생 | 캐릭터 이름 위 칭호 표시 |
| 코스튬 | 캐시샵 | 장비 외형 위에 오버레이 |

> **[디자이너 의견]** 코스튬이 장비 외형 위에 오버레이되면 강화 이펙트와 충돌할 수 있다. 코스튬 착용 시 강화 이펙트 표시 여부를 유저가 선택할 수 있도록 옵션을 제공해야 한다.
>
> **[프로그래머 의견]** 코스튬 오버레이 + 강화 이펙트 + 드래곤 각성 외형이 동시 적용되면 렌더링 부하가 높아진다. 이펙트 레이어 우선순위를 정해 최대 동시 표시 개수를 제한해야 한다.
>
> **[기획자 결정]** 이펙트 우선순위: 드래곤 각성 > 강화 이펙트 > 코스튬. 유저가 개별 토글 가능하도록 옵션 UI 제공. 타인 캐릭터의 이펙트는 거리에 따라 자동 간소화.

## 5. 파티 시스템

### 5.1 파티 구성

| 항목 | 내용 |
|------|------|
| 최대 인원 | 8명 |
| 역할 구분 | 탱커 / 딜러 / 힐러 (강제 아님) |
| 파티 찾기 | 자동 매칭 (던전) + 수동 모집 (필드) |
| 경험치 분배 | 균등 분배 (인원 보정: 2인 ×1.2, 4인 ×1.5, 8인 ×1.8) |
| 아이템 분배 | 주사위/순번/파티장 지정 중 선택 |

### 5.2 레이드 그룹

- 최대 **4파티 (32명)** 연합
- 레이드 보스 및 공성전에서 사용
- 레이드 내 파티 간 버프 공유 없음 (파티 내에서만 유효)

> **[기획자 의견]** 파티 인원 보정 경험치는 '파티 사냥이 솔로보다 이득'인 구조를 만들어 자연스럽게 사회적 플레이를 유도한다. 리니지의 핵심이었던 '사냥터에서의 만남'을 재현하는 장치다.
